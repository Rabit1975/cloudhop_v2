### Loading State

This example illustrates the loading state of the GameList component, showing skeleton cards while data is being fetched.

```tsx
<GameList loading loadingItemCount={4} />
```

### Empty State

This example presents the empty state of the GameList component, indicating that no games are available to display.

```tsx
<GameList 
  games={[]} 
  emptyTitle="No games found" 
  emptyMessage="We could not find any games matching your criteria. Try adjusting your filters." 
/>
