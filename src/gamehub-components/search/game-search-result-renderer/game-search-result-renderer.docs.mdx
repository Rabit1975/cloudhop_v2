---

A component designed to render an individual game search result, showcasing its title, description, thumbnail, and platform. The entire component functions as a clickable card, navigating users to the associated game URL.

### Displaying a single game result

This example shows the rendering of a single game search result with a title, description, thumbnail, and platform. The entire card is clickable, navigating to the games URL.

```js
<GameSearchResultRenderer
  result={{
    id: 'game-1',
    content: {
      title: 'Cyber Odyssey',
      description: 'Dive into the neon-soaked streets of the digital frontier. Explore a vast open world filled with cybernetic enhancements, rogue AI, and high-stakes corporate espionage. Experience the future of gaming with immersive ray-traced graphics and haptic feedback integration.',
      thumbnail: 'https://storage.googleapis.com/bit-generated-images/images/image_modern_video_game_thumbnail__d_0_1770834022251.png',
      platform: 'Unity',
      url: '/gamehub/cyber-odyssey',
    },
  }}
/>
```
### Handling results without a platform
This example demonstrates how the component gracefully handles game results where no platform information is provided. The platform badge will not be displayed.
```js
<GameSearchResultRenderer
  result={{
    id: 'game-2',
    content: {
      title: 'Mysterious Journey',
      description: 'An epic adventure through unknown lands, where ancient ruins hide untold secrets and dangerous creatures roam free.',
      thumbnail: 'https://storage.googleapis.com/bit-generated-images/images/image_modern_video_game_thumbnail__d_0_1770834022251.png',
      platform: undefined, // No platform specified
      url: '/gamehub/mysterious-journey',
    },
  }}
/>
```
