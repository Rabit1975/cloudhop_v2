---
description: A component to display individual player search results.
labels: ['player', 'search', 'ui']
---

This component displays a single player search result, showing the player's avatar, display name, username, and status message. The entire card is clickable and navigates to the player's profile page.

### Basic Player Result

This example shows a basic player search result with a display name, username, avatar, and status message.

```tsx
<PlayerSearchResultRenderer
  result={{
    id: 'res-1',
    relevanceScore: 0.95,
    content: {
      id: 'player-u-1',
      type: 'player',
      title: 'Cosmic Traveler',
      url: '/profile?userId=u-1',
      userId: 'u-1',
      username: 'cosmic_traveler',
      displayName: 'Cosmic Traveler',
      avatarUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_avatar_for_a__0_1770833865266.png',
      statusMessage: 'Drifting through the nebula...',
      presenceStatus: 'online',
    },
  }}
/>
```
