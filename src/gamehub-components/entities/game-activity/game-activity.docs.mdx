A component for a GameActivity entity, supporting serialization and deserialization of GameActivity objects. It can be used from both backend and frontend.

## Get started

Install and use the component in your code:

```ts
const activity = GameActivity.from({
  userId: 'user123',
  gameId: 'game456',
  type: 'played',
  timestamp: new Date().toISOString(),
  details: {
    score: 1200
  }
});
```

## Usage examples

### Creating a new game activity
Instantiate a GameActivity object for a user playing a game.

```ts
const newPlayedActivity = GameActivity.from({
  userId: 'userABC',
  gameId: 'gameXYZ',
  type: 'played',
  timestamp: new Date().toISOString(),
  details: {
    score: 5000
  }
});
console.log(newPlayedActivity.userId);
// expected output: userABC
```

### Recording an achievement
Instantiate a GameActivity object for a user achieving something in a game.

```ts
const newAchievementActivity = GameActivity.from({
  userId: 'userDEF',
  gameId: 'gameMNO',
  type: 'achieved',
  timestamp: new Date().toISOString(),
  details: {
    achievementId: 'first_boss_kill'
  }
});
console.log(newAchievementActivity.details?.achievementId);
// expected output: first_boss_kill
```

### Converting activity to a plain object
Convert a GameActivity instance into a serializable plain object.

```ts
const activity = GameActivity.from({
  userId: 'userGHI',
  gameId: 'gamePQR',
  type: 'streamed',
  timestamp: new Date().toISOString(),
  details: {
    streamId: 'live_stream_123'
  }
});
const plainObject = activity.toObject();
console.log(plainObject.type);
// expected output: streamed
