### Creating a new user

This example demonstrates how to create a new User instance with specific properties and then convert it to a plain object for serialization or data transfer.

```ts
const newUser = new User(
  'user-123',
  'johndoe',
  'John Doe',
  'john.doe@example.com',
  [UserRole.User]
);
console.log('New User:', newUser);

const plainObject = newUser.toObject();
console.log('Plain User Object:', plainObject);

// You can also create a User from a plain object
const userFromJson = User.from(plainObject);
console.log('User from JSON:', userFromJson);
```

### Checking user roles
This example shows how to verify if a user possesses a particular role using the hasRole method.

```ts
const adminUser = new User(
  'admin-456',
  'systemadmin',
  'System Administrator',
  'admin@example.com',
  [UserRole.Admin, UserRole.User]
);
console.log('Is Admin:', adminUser.hasRole(UserRole.Admin));
console.log('Is Guest:', adminUser.hasRole(UserRole.Guest));
```
