### Default Features Section
Displays the FeaturesSection component with its default content.

```jsx
<FeaturesSection />
```
### Custom Title and Subtitle
Shows how to customize the section title and subtitle.
```jsx
<FeaturesSection
  title="Explore CloudHop Capabilities"
  subtitle="Discover the core capabilities of our innovative platform designed for modern communication."
/>
```
