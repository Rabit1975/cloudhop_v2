A versatile and responsive header component for the CloudHop platform. It supports displaying a logo, navigation links, call to actions, and user-specific menu items. The component adjusts its layout for different screen sizes and allows for content customization.

### Basic Guest Header

This example demonstrates a header for unauthenticated users, displaying a platform name, a set of relevant links, and a theme toggler action.

```tsx
<MemoryRouter>
  <CloudrabbitTheme>
    <Header
      name="CloudHop"
      links={SAMPLE_LINKS}
      authenticated={false}
      actions={[ThemeToggler]}
    />
  </CloudrabbitTheme>
</MemoryRouter>
```

### Authenticated User Header

This example shows a header for authenticated users, including platform name with a slogan, user-specific navigation links, and additional actions like notifications and a user avatar.

```tsx
<MemoryRouter>
  <CloudrabbitTheme>
    <Header
      name="CloudHop"
      slogan="Beta"
      links={SAMPLE_LINKS}
      authenticated={true}
      actions={[ThemeToggler, NotificationIcon]}
      userBarItems={[UserAvatar]}
    />
  </CloudrabbitTheme>
</MemoryRouter>
```

### Header with Custom Center Content

This example illustrates how to inject custom content, such as a search bar, into the central part of the header.

```tsx
<MemoryRouter>
  <CloudrabbitTheme>
    <Header
      name="CloudHop"
      links={SAMPLE_LINKS}
      authenticated={true}
      actions={[ThemeToggler]}
      userBarItems={[UserAvatar]}
    >
      <div style={{
        maxWidth: '400px',
        width: '100%',
        position: 'relative',
        display: 'flex',
        alignItems: 'center'
      }}>
        <input
          placeholder="Search..."
          style={{
            width: '100%',
            padding: '8px 16px',
            borderRadius: '20px',
            border: '1px solid var(--colors-border-default)',
            backgroundColor: 'var(--colors-surface-secondary)',
            color: 'var(--colors-text-primary)'
          }}
        />
      </div>
    </Header>
  </CloudrabbitTheme>
</MemoryRouter>
