A versatile panel card component designed for displaying modular content in the CloudHop platform. It supports various visual styles, includes a title, optional icon, actions, and a footer, and can display a loading state.

### Basic panel with content and actions

This example demonstrates a panel card with a title, an icon, and some interactive actions in the header. The main content area contains a list of items.

```tsx
import { PanelCard } from '@cloudhop/ui.panel-card';
import { SettingsIcon } from '@cloudhop/ui.icons.settings';
import { Button } from '@cloudhop/ui.button';
const MyPanel = () => (
  <div style={{ maxWidth: '400px', height: '300px' }}>
    <PanelCard
      title="My Custom Module"
      icon={<SettingsIcon />}
      actions={
        <>
          <Button variant="ghost">Edit</Button>
          <Button>Save</Button>
        </>
      }
    >
      <div style={{ padding: '16px', display: 'flex', flexDirection: 'column', gap: '10px' }}>
        <p>This is some content for the module.</p>
        <p>You can put anything here, like settings or data displays.</p>
      </div>
    </PanelCard>
  </div>
);
```

### Nebula styled panel with footer

This example showcases the nebula variant, which provides a translucent, glassy appearance suitable for background-rich sections. It also includes a footer area for additional controls or information.

```tsx
import { PanelCard } from '@cloudhop/ui.panel-card';
import { ChatIcon } from '@cloudhop/ui.icons.chat';
import { Button } from '@cloudhop/ui.button';
const ChatPanel = () => (
  <div style={{ 
    padding: '40px', 
    maxWidth: '400px', 
    height: '500px',
    background: 'linear-gradient(135deg, #0f172a 0%, #1e293b 100%)'
  }}>
    <PanelCard
      title="# general"
      variant="nebula"
      icon={<ChatIcon />}
      actions={<Button variant="ghost">View Thread</Button>}
      footer={
        <input 
          placeholder="Message #general..." 
          style={{ 
            width: '100%', 
            padding: '12px', 
            borderRadius: '8px',
            border: 'none',
            backgroundColor: 'rgba(30, 41, 59, 0.5)',
            color: 'white',
            boxSizing: 'border-box'
          }} 
        />
      }
    >
      <div style={{ display: 'flex', flexDirection: 'column', gap: '16px', overflowY: 'auto', flex: 1, padding: '16px' }}>
        <div style={{ display: 'flex', gap: '12px' }}>
          <div style={{ width: '32px', height: '32px', borderRadius: '50%', backgroundColor: 'blue', flexShrink: 0 }} />
          <div>
            <div style={{ fontSize: '12px', fontWeight: 'bold', marginBottom: '4px', color: 'white' }}>Neo</div>
            <div style={{ fontSize: '14px', color: '#ccc', lineHeight: 1.4 }}>
              Has anyone checked out the new Music Studio module? The nebula effects are insane!
            </div>
          </div>
        </div>
      </div>
    </PanelCard>
  </div>
);
```

### Panel with a loading state

This example demonstrates how the panel card behaves when it is in a loading state. The content area is replaced with a loading indicator.

```tsx
import { PanelCard } from '@cloudhop/ui.panel-card';
import { CalendarIcon } from '@cloudhop/ui.icons.calendar';
const LoadingPanel = () => (
  <div style={{ padding: '40px', maxWidth: '300px', height: '250px' }}>
    <PanelCard
      title="Upcoming Events"
      icon={<CalendarIcon />}
      loading
      variant="clean"
    />
  </div>
);
