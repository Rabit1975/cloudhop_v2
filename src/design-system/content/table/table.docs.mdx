### Basic Table

This example demonstrates a basic table structure with simple columns and data.

```jsx
const columns = [
  { key: 'name', title: 'Server Name' },
  { key: 'region', title: 'Region' },
  { key: 'players', title: 'Active Users' },
  { key: 'status', title: 'Status' },
];
const data = [
  { id: '1', name: 'CloudHop HQ', region: 'NA-East', players: '1,240', status: 'Online' },
  { id: '2', name: 'Music Lounge', region: 'EU-West', players: '856', status: 'Online' },
  { id: '3', name: 'Gaming Arena', region: 'Asia-Pacific', players: '3,402', status: 'Busy' },
];
return (
  <div style={{ padding: '24px', backgroundColor: '#f5f5f5', borderRadius: '8px' }}>
    <Table columns={columns} data={data} />
  </div>
);
```
### Custom Column Rendering and Actions
This example shows how to customize column rendering using the render prop and add interactive elements.
