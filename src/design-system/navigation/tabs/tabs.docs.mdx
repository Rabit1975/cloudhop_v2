### Basic Navigation

This example demonstrates a basic tab navigation setup with several links. The active tab is automatically highlighted based on the current URL path.

```tsx
const tabs = [
  { label: 'HopHub', href: '/hophub' },
  { label: 'HopMeets', href: '/hopmeets' },
  { label: 'Music', href: '/music' },
  { label: 'Spaces', href: '/spaces' },
  { label: 'GameHub', href: '/gamehub' },
];
<Tabs tabs={tabs} />
```

### Active Tab Indication
This example shows how the component visually indicates the currently active tab. The activity tab is pre selected to show the styling.
```tsx
const tabs = [
  { label: 'Overview', href: '/overview' },
  { label: 'Activity', href: '/activity' },
  { label: 'Settings', href: '/settings' },
];
// In a real application, the active tab would be determined by the router.
// For demonstration, assume /activity is the current path.
<Tabs tabs={tabs} />
```
