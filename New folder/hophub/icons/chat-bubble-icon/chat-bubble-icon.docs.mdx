### Render a basic chat bubble icon

This shows the default chat bubble icon without any custom properties.

```tsx
<ChatBubbleIcon />
```

### Change the icon color
You can easily customize the color of the chat bubble icon using the color property.
```tsx
<ChatBubbleIcon color="var(--colors-primary-default)" />
<ChatBubbleIcon color="var(--colors-status-positive-default)" />
<ChatBubbleIcon color="var(--colors-text-secondary)" />
```

### Set the icon size
The size property allows you to control the width and height of the icon.
```tsx
<ChatBubbleIcon size={16} />
<ChatBubbleIcon size={24} />
<ChatBubbleIcon size={32} />
<ChatBubbleIcon size={48} />
