A data entity for a HopHub direct message conversation, designed to support serialization and deserialization. It can be used in both frontend and backend contexts to manage direct message data.

## Get started

### Creating a new Direct Message instance

This example demonstrates how to create a `DirectMessage` instance from a plain object using the static `from` method. This is useful when you receive data from an API or a database.

```typescript
import { DirectMessage } from './direct-message.js';
const directMessageData = {
  id: 'dm-abc-123',
  participantIds: ['user-101', 'user-102'],
  createdAt: '2023-10-26T10:00:00Z',
  updatedAt: '2023-10-26T10:30:00Z',
};
const directMessage = DirectMessage.from(directMessageData);
console.log(directMessage.id);
console.log(directMessage.participantIds);
```

### Serializing a Direct Message instance
This example shows how to convert a `DirectMessage` instance back into a plain object using the `toObject` method. This is often needed when sending data to an API or storing it.
```typescript
import { DirectMessage } from './direct-message.js';
const initialDirectMessage = new DirectMessage(
  'dm-def-456',
  ['user-201', 'user-202'],
  '2023-10-25T09:00:00Z',
  '2023-10-25T09:15:00Z'
);
const plainObject = initialDirectMessage.toObject();
console.log(plainObject.id);
console.log(plainObject.participantIds);
```

## Mock data
### Using mock data for development
You can use mock direct message data for development and testing purposes. This example demonstrates how to generate a list of mock `DirectMessage` instances.
```typescript
import { mockDirectMessages } from './direct-message.mock.js';
const messages = mockDirectMessages();
messages.forEach(message => {
  console.log(`DM ID: ${message.id}, Participants: ${message.participantIds.join(', ')}`);
});
