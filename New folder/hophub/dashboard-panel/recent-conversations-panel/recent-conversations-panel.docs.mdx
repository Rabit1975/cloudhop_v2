### Render a basic panel

This example shows the basic rendering of the recent conversations panel with a predefined list of conversations.

```tsx

    lastMessage: {
      id: 'm1',
      conversationId: 'c1',
      senderId: 'u2',
      text: 'Has anyone seen the new design?',
      type: 'TEXT',
      createdAt: new Date().toISOString(),
    }
  },
  {
    id: 'c2',
    type: 'DM',
    name: 'Alice',
    unreadCount: 0,
    imageUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_illustration__0_1770835145620.png',
    lastMessage: {
      id: 'm2',
      conversationId: 'c2',
      senderId: 'u3',
      text: 'Sure, lets sync later.',
      type: 'TEXT',
      createdAt: new Date(Date.now() - 3600000).toISOString(),
    }
  },
  {
    id: 'c3',
    type: 'GROUP',
    name: 'Project Alpha',
    unreadCount: 1,

<div style={{ height: 500, width: 350, padding: 24, border: '1px solid var(--colors-border-primary)', borderRadius: 'var(--borders-radius-medium)' }}>

### Display a loading state

This example demonstrates the panel displaying a loading state when conversations are being fetched.

```tsx
<div style={{ height: 500, width: 350, padding: 24, border: '1px solid var(--colors-border-primary)', borderRadius: 'var(--borders-radius-medium)' }}>
  {/* When the 'conversations' prop is not provided, the component uses its internal loading mechanism */}
  <RecentConversationsPanel />
</div>

### Display an empty state

This example illustrates the panel displaying an empty state when there are no recent conversations.

```tsx
