A presence indicator to display the user status within HopHub.

### Basic Usage

Displays the presence indicator for various user statuses.

```tsx
<HopHubPresenceIndicator status="online" />
<HopHubPresenceIndicator status="away" />
<HopHubPresenceIndicator status="busy" />
<HopHubPresenceIndicator status="offline" />
```
### Customizing Size
Demonstrates how to adjust the size of the presence indicator using the size prop.
```tsx
<HopHubPresenceIndicator status="online" size="small" />
<HopHubPresenceIndicator status="away" size="medium" />
<HopHubPresenceIndicator status="busy" size="large" />
```
### Integration with Avatar
Illustrates placing the presence indicator on top of an avatar to show user status.
