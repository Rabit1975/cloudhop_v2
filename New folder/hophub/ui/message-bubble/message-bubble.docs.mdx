### Received Message

This example shows a basic received message from another user with text content.

```tsx
const otherUser = {
  id: 'user-123',
  displayName: 'Alice',
  imageUrl: 'https://static.bit.dev/illustrations/bit-logo-green.svg',
};
const message = {
  id: 'msg-rec-1',
  senderId: 'user-123',
  text: 'Hello there! How are you doing today?',
  createdAt: new Date().toISOString(),
};
return (
  <div style={{ padding: '24px', maxWidth: '600px' }}>
    <MessageBubble message={message} sender={otherUser} isSelf={false} />
  </div>
);
```

### Message with Media
This example demonstrates a received message that includes both text and an image attachment.
```tsx
const otherUser = {
  id: 'user-456',
  displayName: 'Bob',
  imageUrl: 'https://static.bit.dev/illustrations/bit-logo-green.svg',
};
const message = {
  id: 'msg-media-1',
  senderId: 'user-456',
  text: 'Check out this cool image I found!',
  mediaAttachments: [
    {
      id: 'media-img-1',
      url: 'https://storage.googleapis.com/bit-generated-images/images/image_collection_of_diverse_and_mode_0_1770833880700.png',
      type: 'image/png',
      filename: 'cool-image.png',
    },
  ],
  createdAt: new Date().toISOString(),
};
return (
  <div style={{ padding: '24px', maxWidth: '600px' }}>
    <MessageBubble message={message} sender={otherUser} isSelf={false} />
  </div>
);
```
### Conversation Flow
This example displays a sequence of sent and received messages, illustrating a typical chat conversation.
```tsx
const currentUser = {
  id: 'user-self',
  displayName: 'You',
  imageUrl: 'https://static.bit.dev/illustrations/bit-logo-black.svg',
};
const otherUser = {
  id: 'user-789',
  displayName: 'Charlie',
  imageUrl: 'https://static.bit.dev/illustrations/bit-logo-green.svg',
};
const msg1 = {
  id: 'conv-1',
  senderId: otherUser.id,
  text: 'Hey there! How is the new CloudHop project coming along?',
  createdAt: new Date(Date.now() - 600000).toISOString(),
};
const msg2 = {
  id: 'conv-2',
  senderId: currentUser.id,
  text: 'It is going great! We just finished the initial design for the HopHub chat interface.',
  createdAt: new Date(Date.now() - 500000).toISOString(),
};
const msg3 = {
  id: 'conv-3',
  senderId: otherUser.id,
  text: 'Awesome! Any sneak peeks? I am excited to see it.',
  mediaAttachments: [
    {
      id: 'media-img-2',
      url: 'https://storage.googleapis.com/bit-generated-images/images/image_collection_of_diverse_and_mode_0_1770833880700.png',
      type: 'image/png',
      filename: 'design-preview.png',
    },
  ],
  createdAt: new Date(Date.now() - 400000).toISOString(),
};
const msg4 = {
  id: 'conv-4',
  senderId: currentUser.id,
  text: 'For sure! The team has done an amazing job with the nebula background.',
  createdAt: new Date(Date.now() - 300000).toISOString(),
};
return (
  <div style={{ padding: '24px', maxWidth: '600px', display: 'flex', flexDirection: 'column', gap: '16px' }}>
    <MessageBubble message={msg1} sender={otherUser} isSelf={false} />
    <MessageBubble message={msg2} sender={currentUser} isSelf={true} />
    <MessageBubble message={msg3} sender={otherUser} isSelf={false} />
    <MessageBubble message={msg4} sender={currentUser} isSelf={true} />
  </div>
);
