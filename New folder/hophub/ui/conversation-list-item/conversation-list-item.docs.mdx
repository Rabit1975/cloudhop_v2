---
description: A component to display a single conversation in a list, showing its name, last message snippet, and unread count.
labels: ['conversation', 'list', 'item', 'chat', 'ui']
---

A ConversationListItem component efficiently displays key information about a single conversation,
such as its name, the latest message snippet, and any unread message count. It is designed to be
clickable, allowing users to navigate directly to the conversation or trigger a custom action.

### Displaying a Standard Conversation

This example demonstrates how to render typical conversation list items, highlighting names,
last messages, and unread badges. It also shows how to mark an item as active.

### Conversation with an Unread Badge

This example demonstrates a conversation item displaying an unread message count badge, visually indicating new messages awaiting attention.

```tsx
// Example data for demonstration
const mockUnreadConversation = {
  id: 'unread-conv-id',
  type: 'DM',
  name: 'Important Client',
  members: [],
  createdAt: new Date().toISOString(),
  updatedAt: new Date().toISOString(),
  lastMessage: {
    id: 'msg-unread',
    conversationId: 'unread-conv-id',
    senderId: 'client-user',
    text: 'Please check the contract attached.',
    type: 'TEXT',
    createdAt: new Date(Date.now() - (1000 * 60 * 15)).toISOString(), // 15 minutes ago
  },
  unreadCount: 12,
};
<div style={{ padding: '24px', maxWidth: '400px', backgroundColor: 'var(--colors-surface-background)', borderRadius: '12px' }}>
  <ConversationListItem
    conversation={mockUnreadConversation as any}
    currentUserId="user-1"
  />
</div>
