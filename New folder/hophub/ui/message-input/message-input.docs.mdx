A `MessageInput` component provides a text input field, an attachment button for media, and a send button, designed for use in chat applications.

### Basic Message Input

Shows the message input in a typical chat conversation flow. Users can type and send messages, which then appear in a message list.

```tsx
import { useState } from 'react';
import { MessageInput } from './message-input.js';
const mockMessages = [
  { id: '1', user: 'Alice', text: 'Hey! Did you check out the new Music Studio in CloudHop?' },
  { id: '2', user: 'Bob', text: 'Yeah, it integrates perfectly with my YouTube playlists.' },
];
export default function BasicMessageInputExample() {
  const [messages, setMessages] = useState(mockMessages);
  const handleSend = (text) => {
    setMessages((prev) => [
      ...prev,
      { id: Date.now().toString(), user: 'You', text },
    ]);
  };
  return (
    <div
      style={{
        display: 'flex',
        flexDirection: 'column',
        height: '400px',
        maxWidth: '600px',
        border: '1px solid var(--colors-border-subtle)',
        borderRadius: 'var(--borders-radius-medium)',
        overflow: 'hidden',
        backgroundColor: 'var(--colors-surface-background)',
      }}
    >
      <div style={{ flex: 1, padding: 'var(--spacing-medium)', overflowY: 'auto' }}>
        {messages.map((msg) => (
          <div
            key={msg.id}
            style={{
              marginBottom: 'var(--spacing-medium)',
              textAlign: msg.user === 'You' ? 'right' : 'left',
            }}
          >
            <div
              style={{
                fontSize: 'var(--typography-sizes-caption-default)',
                color: 'var(--colors-text-secondary)',
                marginBottom: 'var(--spacing-xs)',
              }}
            >
              {msg.user}
            </div>
            <div
              style={{
                display: 'inline-block',
                padding: 'var(--spacing-small) var(--spacing-medium)',
                borderRadius: 'var(--borders-radius-medium)',
                backgroundColor:
                  msg.user === 'You'
                    ? 'var(--colors-primary-default)'
                    : 'var(--colors-surface-secondary)',
                color:
                  msg.user === 'You'
                    ? 'var(--colors-primary-contrast)'
                    : 'var(--colors-text-primary)',
              }}
            >
              {msg.text}
            </div>
          </div>
        ))}
      </div>
      <MessageInput onSendMessage={handleSend} />
    </div>
  );
}
```

### Disabled Input

Demonstrates the message input when it is disabled, preventing users from typing or sending messages. This can be useful for read-only channels or when user permissions restrict sending.

```tsx
import { MessageInput } from './message-input.js';
export default function DisabledMessageInputExample() {
  return (
    <div style={{ padding: 'var(--spacing-large)', maxWidth: '600px', backgroundColor: 'var(--colors-surface-background)' }}>
      <h3 style={{ color: 'var(--colors-text-primary)', marginBottom: 'var(--spacing-medium)' }}>
        Read-only Channel
      </h3>
      <MessageInput disabled placeholder="You do not have permission to send messages to this channel." />
    </div>
  );
}
```

### Dark Theme

Illustrates how the message input component adapts its styling when used within a dark theme environment.

```tsx
import { MessageInput } from './message-input.js';
import { CloudrabbitTheme } from '@cloudrabbit/design.cloudrabbit-theme';
export default function DarkThemeMessageInputExample() {
  return (
    <CloudrabbitTheme initialTheme="dark">
      <div
        style={{
          padding: 'var(--spacing-large)',
          maxWidth: '600px',
          backgroundColor: 'var(--colors-surface-background)',
          height: '100%',
          minHeight: '200px',
          display: 'flex',
          alignItems: 'center',
        }}
      >
        <MessageInput placeholder="Message #general" />
      </div>
    </CloudrabbitTheme>
  );
}
