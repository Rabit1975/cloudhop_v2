A `SearchResultCard` visually presents a search result entity including its title, description, thumbnail if available, and type. The entire card is clickable to navigate to the URL associated with the content.

### Basic Search Result Card

Illustrates the default appearance of a search result card with minimal information. This example uses the default mock data for the component.

```tsx
  <SearchResultCard />
```

### Search Result Card with all properties

Displays a search result card populated with a title, description, image, and type badge. The entire card is clickable.
```tsx
  <SearchResultCard
    result={{
      id: 'example-1',
      title: 'CloudHop Channel Introduction',
      description: 'Discover the features of CloudHop channels and how to create your own community space.',
      type: 'Channel',
      imageUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_network_backg_0_1770833860674.png',
      link: '/channels/intro'
    }}
  />
```

Shows how to render several search result cards in a responsive grid layout. This demonstrates how the component behaves when used in a list of results.
```tsx

      link: '/channels/general'

      title: 'Cyber Nebula',
