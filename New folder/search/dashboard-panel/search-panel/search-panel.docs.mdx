A dedicated search panel for CloudHop, allowing users to quickly find information across the platform. It features a prominent input field, a search icon, and navigates to a search results page upon submission.

## Component usage

The search panel provides a centralized way to initiate searches within the CloudHop platform.

```tsx
<SearchPanel />
```
### Basic Search Panel
This example shows the default appearance and functionality of the SearchPanel component. It uses the default placeholder text and directs to a search page.
```tsx
<SearchPanel />
```
### Search Panel with Custom Placeholder
You can customize the placeholder text within the search input to provide more specific guidance to users. This is useful for different contexts or sub-sections of the platform.
```tsx
<SearchPanel placeholder="Search channels, messages, files..." />
```
