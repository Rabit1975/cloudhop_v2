A featured search section designed for a homepage or dashboard to encourage users to start their search experience. It includes a title, a description, and an integrated search input that directs to the `/search` page upon submission.

### Basic Usage

A basic featured search section with default title and subtitle.

```tsx
<FeaturedSearchSection />
```
### Custom Content
A featured search section with a custom title, subtitle, and caption.
```tsx
<FeaturedSearchSection
  title="Find Your Squad"
  subtitle="Connect with gamers, developers, and creators instantly."
  caption="Community Search"
/>
```
### Full Width Display
A featured search section styled to take full width with default content.
```tsx
<FeaturedSearchSection />
