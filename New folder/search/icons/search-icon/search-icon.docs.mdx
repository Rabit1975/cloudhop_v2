A standard search icon component, perfect for integrating search functionality across CloudHop.

### Basic usage

Render the search icon with its default size and color.

```tsx
<CloudrabbitTheme>
  <div style={{ padding: '24px' }}>
    <SearchIcon />
  </div>
</CloudrabbitTheme>
```
### Customizing size
Adjust the size of the search icon using the size prop.
```tsx
<CloudrabbitTheme>
  <div style={{ 
    display: 'flex', 
    alignItems: 'center', 
    gap: '24px', 
    padding: '24px',
    color: 'var(--colors-text-primary)' 
  }}>
    <SearchIcon size={16} />
    <SearchIcon size={24} />
    <SearchIcon size={32} />
  </div>
</CloudrabbitTheme>
```
### Customizing color
Change the color of the search icon using the color prop.
```tsx
<CloudrabbitTheme>
  <div style={{ 
    display: 'flex', 
    alignItems: 'center', 
    gap: '24px', 
    padding: '24px',
    backgroundColor: 'var(--colors-surface-background)' 
  }}>
    <SearchIcon color="var(--colors-primary-default)" />
    <SearchIcon color="var(--colors-secondary-default)" />
    <SearchIcon color="var(--colors-text-secondary)" />
  </div>
</CloudrabbitTheme>
