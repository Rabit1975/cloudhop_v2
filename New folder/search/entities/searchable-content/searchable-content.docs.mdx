A component that defines the structure for content to be included in search results,
supporting serialization and deserialization. This entity can be used across frontend
and backend systems to ensure consistent data handling for searchable items.


### Creating a SearchableContent item

This example demonstrates how to create a SearchableContent instance from a plain object, providing all necessary properties including an optional thumbnail and keywords.

```typescript
const mySearchableContent = SearchableContent.from({
  id: 'doc-001',
  type: 'document',
  title: 'CloudHop User Guide',
  description: 'A comprehensive guide for new CloudHop users covering all features.',
  url: '/docs/user-guide',
  thumbnail: 'https://cloudhop.com/thumbnails/user-guide.png',
  keywords: ['guide', 'help', 'onboarding', 'documentation'],
});
console.log(mySearchableContent.title);
// Expected output: CloudHop User Guide
```

### Converting a SearchableContent item to a plain object

This example shows how to convert a SearchableContent instance back into a plain JavaScript object, useful for data transfer or storage.

```typescript
const item = SearchableContent.from({
  id: 'doc-001',
  type: 'document',
  title: 'CloudHop User Guide',
  description: 'A comprehensive guide for new CloudHop users covering all features.',
  url: '/docs/user-guide',
  keywords: ['guide', 'help'],
});
const plainObject = item.toObject();
console.log(plainObject.type);
// Expected output: document
console.log(plainObject.keywords);
// Expected output: ['guide', 'help']
```
