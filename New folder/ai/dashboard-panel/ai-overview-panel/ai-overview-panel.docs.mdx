An overview panel designed for displaying AI related information or quick access to AI features within a dashboard. It includes a title, a brief description, an AI icon, and a call to action button.

### Basic Usage

This example shows the AI Overview Panel with its default title and description.

```tsx
<AiOverviewPanel />
```
### Custom Content
This example demonstrates how to customize the title and description of the AI Overview Panel.
```tsx
<AiOverviewPanel
  title="AI Assistant"
  description="Your intelligent companion for generating content, summarizing data, and more."
/>
```
### Applying Custom Styles
This example illustrates how to apply custom styles to the AI Overview Panel using the className prop.
```tsx
<AiOverviewPanel
  className="custom-ai-panel"
  title="Creative AI"
  description="Generate stunning visuals and unique text content with our creative AI tools."
/>
