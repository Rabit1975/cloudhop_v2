### Creating AI settings

This example demonstrates how to create an AI settings object from a plain JavaScript object.

```ts
// Assume AiSettings is imported or available in the context.
const openAISettings = AiSettings.from({
  defaultProvider: 'openai',
  defaultModel: 'gpt-4',
  apiKey: 'sk-your-openai-key',
  temperature: 0.7,
  maxTokens: 2048,
});
const anthropicSettings = AiSettings.from({
  defaultProvider: 'anthropic',
  defaultModel: 'claude-3-opus',
  apiKey: 'sk-your-anthropic-key',
  temperature: 0.5,
  maxTokens: 4096,
});
console.log(openAISettings);
console.log(anthropicSettings);
```
### Accessing AI settings properties
This example shows how to retrieve individual settings such as the default provider and API key.
```ts
// Assume AiSettings is imported or available in the context.
const aiSettings = AiSettings.from({
  defaultProvider: 'openai',
  defaultModel: 'gpt-4',
  apiKey: 'sk-mock-key-12345',
  temperature: 0.7,
  maxTokens: 2048,
});
console.log(`Default Provider: ${aiSettings.defaultProvider}`);
console.log(`API Key: ${aiSettings.apiKey}`);
console.log(`Max Tokens: ${aiSettings.maxTokens}`);
```
### Serializing AI settings
This example demonstrates how to convert an AI settings object back into a plain object for storage or transmission.
```ts
// Assume AiSettings is imported or available in the context.
const aiSettings = new AiSettings(
  'openai',
  'gpt-4',
  'sk-mock-key-12345',
  0.7,
  2048
);
const plainObject = aiSettings.toObject();
console.log(plainObject);
/*
Output:
{
  defaultProvider: 'openai',
  defaultModel: 'gpt-4',
  apiKey: 'sk-mock-key-12345',
  temperature: 0.7,
  maxTokens: 2048,
}
*/
