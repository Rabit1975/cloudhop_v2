### Basic AI Icon

A basic AI icon with default size and color.

```tsx
<AiIcon />
```
### AI Icon with custom sizes
The AI icon can be rendered in various sizes using the size prop.
```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: '24px' }}>
  <AiIcon size={16} />
  <AiIcon size={24} />
  <AiIcon size={32} />
  <AiIcon size={48} />
</div>
```
### AI Icon with custom colors
The AI icon can be colored using the color prop which accepts CSS color values.
```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: '24px' }}>
  <AiIcon color="blue" />
  <AiIcon color="green" />
  <AiIcon color="purple" />
  <AiIcon color="orange" />
</div>
