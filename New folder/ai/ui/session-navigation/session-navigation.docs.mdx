---
description: A navigation component for managing chat sessions and displaying recent chat history.
labels: ['navigation', 'session', 'chat', 'ui']
---

This component displays a list of recent chat sessions, allowing users to navigate between them and start new conversations. It is designed to be integrated into a larger application, providing a persistent sidebar for AI chat history.

### Basic Session History Display
A simple example showing the session navigation with a list of recent chat sessions. This component allows users to browse through their chat history.

```tsx
const mockSessions = [
  { id: 'session-1', name: 'Project Alpha Discussion', createdAt: new Date(Date.now() - 3600000).toISOString() },
  { id: 'session-2', name: 'Daily Standup Notes', createdAt: new Date(Date.now() - 86400000).toISOString() },
  { id: 'session-3', name: 'CloudHop Feature Brainstorm', createdAt: new Date(Date.now() - 2 * 86400000).toISOString() },
  { id: 'session-4', name: 'Marketing Campaign Review', createdAt: new Date(Date.now() - 3 * 86400000).toISOString() },
];
<div style={{ width: '300px', height: '600px', border: '1px solid var(--colors-border-subtle)' }}>
  <SessionNavigation mockData={mockSessions} />
</div>
```
### Empty Chat History
This example demonstrates the appearance of the session navigation when there are no previous chat sessions available. It displays a message indicating an empty history.
```tsx
<div style={{ width: '300px', height: '600px', border: '1px solid var(--colors-border-subtle)' }}>
  <SessionNavigation mockData={[]} />
</div>
```
### Handling New Session Creation
This example shows how to integrate a callback function for the new session button. When the "New Session" button is clicked, the provided `onNewSession` function will be executed.
```tsx
const handleNewSession = () => {
  alert('New session button clicked');
};
<div style={{ width: '300px', height: '600px', border: '1px solid var(--colors-border-subtle)' }}>
  <SessionNavigation onNewSession={handleNewSession} mockData={[]} />
</div>
