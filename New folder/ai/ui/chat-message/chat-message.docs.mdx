---
description: A UI component to display a single chat message, distinguishing between user and AI roles.
labels: ['chat', 'message', 'ui', 'communication']
---

A `ChatMessage` component displays individual messages within a chat interface. It visually distinguishes between messages sent by a user and those from an AI assistant, enhancing readability and user experience. The component includes the message text, an avatar, and metadata such as the sender's name and timestamp.

### Displaying a User Message

This example shows how to render a message sent by a user, including their avatar and message content.

```tsx
import { ChatMessage } from './chat-message.js';
import React from 'react';

const userMessage = {
  id: 'user-msg-1',
  sessionId: 'session-1',
  role: 'user',
  text: 'Hello CloudHop AI, how can I access the GameHub?',
  createdAt: new Date().toISOString(),
  userId: 'user-123',
};
const userAvatar = 'https://picsum.photos/id/1005/50/50';
<ChatMessage message={userMessage} avatarSrc={userAvatar} />
```
### Displaying an AI Message
This example demonstrates how to render a message received from an AI assistant, along with its avatar and response.
```tsx
import { ChatMessage } from './chat-message.js';
import React from 'react';

const aiMessage = {
  id: 'ai-msg-1',
  sessionId: 'session-1',
  role: 'ai',
  text: 'Welcome! You can find the GameHub by navigating to the "GameHub" tab in the main menu.',
  createdAt: new Date().toISOString(),
};
const aiAvatar = 'https://picsum.photos/id/1004/50/50';
<ChatMessage message={aiMessage} avatarSrc={aiAvatar} />
```
### A Simple Conversation Snippet
This example illustrates how to render a short exchange between a user and an AI, showcasing the visual distinction for both roles within a continuous flow.
```tsx
import { ChatMessage } from './chat-message.js';
import React from 'react';

const messageOne = {
  id: 'conv-msg-1',
  sessionId: 'session-2',
  role: 'user',
  text: 'I am looking for creative tools with AI integration. Where are those located?',
  createdAt: new Date(Date.now() - 120000).toISOString(),
  userId: 'user-456',
};
const messageTwo = {
  id: 'conv-msg-2',
  sessionId: 'session-2',
  role: 'ai',
  text: 'Our creative tools, along with AI integration, are available in the "Spaces" section. You will find various exciting features there!',
  createdAt: new Date(Date.now() - 60000).toISOString(),
};
const messageThree = {
  id: 'conv-msg-3',
  sessionId: 'session-2',
  role: 'user',
  text: 'Thank you for the quick response!',
  createdAt: new Date().toISOString(),
  userId: 'user-456',
};
const userAvatar = 'https://picsum.photos/id/1005/50/50';
const aiAvatar = 'https://picsum.photos/id/1004/50/50';
<div>
  <ChatMessage message={messageOne} avatarSrc={userAvatar} />
  <ChatMessage message={messageTwo} avatarSrc={aiAvatar} />
  <ChatMessage message={messageThree} avatarSrc={userAvatar} />
</div>
