### Default and Sized Spinners

This example shows the spinner in its default state and with various predefined sizes.

```tsx
<div style={{ display: 'flex', gap: '32px', alignItems: 'center' }}>
  <Spinner size="small" />
  <Spinner size="medium" />
  <Spinner size="large" />
  <Spinner size="xl" />
</div>
```

### Spinner with Theme Variants
This example demonstrates how to apply different color themes to the spinner using the variant prop.

```tsx
<div style={{ display: 'flex', gap: '32px', alignItems: 'center' }}>
  <Spinner variant="primary" />
  <Spinner variant="secondary" />
  <Spinner variant="neutral" />
  <Spinner variant="nebula" size="large" />
</div>
```

### Spinner Overlay for Content
This example illustrates how the spinner can be used as an overlay to cover content while it is loading. The overlay prop positions the spinner absolutely over its parent.

```tsx
<div style={{ padding: '32px' }}>
  <div style={{
    position: 'relative',
    width: '320px',
    padding: '24px',
    borderRadius: '8px',
    backgroundColor: '#1E293B',
    boxShadow: '0px 4px 12px rgba(0, 0, 0, 0.2)',
    overflow: 'hidden',
    border: '1px solid #334155',
    color: '#E2E8F0'
  }}>
    <div style={{
      height: '24px',
      width: '60%',
      backgroundColor: '#475569',
      marginBottom: '16px',
      borderRadius: '4px'
    }} />
    <div style={{
      height: '16px',
      width: '100%',
      backgroundColor: '#475569',
      marginBottom: '8px',
      borderRadius: '4px'
    }} />
    <div style={{
      height: '16px',
      width: '80%',
      backgroundColor: '#475569',
      borderRadius: '4px',
      marginBottom: '24px'
    }} />
    <div style={{
      height: '40px',
      width: '100px',
      backgroundColor: '#475569',
      borderRadius: '20px'
    }} />
    <Spinner overlay variant="primary" size="large" />
  </div>
</div>
