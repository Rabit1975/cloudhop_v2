A versatile and accessible button component that prioritizes user experience and visual consistency. This component is designed to grab user attention and encourage clicks, incorporating modern design trends.

### Basic usage of the CTA button

This example shows a simple CTA button with default primary appearance.

```jsx
<CtaButton>Join CloudHop</CtaButton>
```

### CTA button appearance variations

This example demonstrates the primary, secondary, and tertiary appearances of the CTA button.

```jsx
<CtaButton appearance="primary">Get Started</CtaButton>
<CtaButton appearance="secondary">View Pricing</CtaButton>
<CtaButton appearance="tertiary">Learn More</CtaButton>
```

### CTA button as a link and disabled state

This example illustrates using the CTA button as a link and how to disable it.

```jsx
<CtaButton href="/signup" appearance="primary">Start Free Trial</CtaButton>
<CtaButton disabled appearance="secondary">Disabled Button</CtaButton>
