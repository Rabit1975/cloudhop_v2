A versatile and accessible button component for triggering actions or navigating. It provides various appearance styles and can act as a link when an `href` is provided.

### Button appearances

Demonstrates the different visual styles available for the button component.

```tsx
<div style={{ display: 'flex', gap: 'var(--spacing-medium)' }}>
  <Button appearance="primary">Create Server</Button>
  <Button appearance="secondary">Join Channel</Button>
  <Button appearance="tertiary">Cancel</Button>
</div>
```
### Navigation buttons
Shows how to use the button component as a navigation link, including external links.
```tsx
<div style={{ display: 'flex', gap: 'var(--spacing-medium)' }}>
  <Button href="/dashboard" appearance="primary">
    Go to Dashboard
  </Button>
  <Button href="https://example.com" external appearance="secondary">
    External Link
  </Button>
</div>
```
### Button states and actions
Illustrates the disabled state of buttons and how to attach click event handlers.
```tsx
<div style={{ display: 'flex', gap: 'var(--spacing-medium)' }}>
  <Button disabled appearance="primary">Saving...</Button>
  <Button disabled appearance="secondary">Unavailable</Button>
  <Button onClick={() => alert('Button clicked!')} appearance="tertiary">
    Click Me
  </Button>
</div>
