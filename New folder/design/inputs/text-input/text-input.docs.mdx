A versatile text input component designed to handle user text entry in forms. It supports various states like disabled, error, and controlled usage, making it suitable for a wide range of applications.

### Basic text input

This example demonstrates a basic text input with a label and a placeholder.

```tsx
<TextInput
  id="username-input"
  label="Username"
  placeholder="Enter your username"
/>
```
### Text input with error message
This example shows how to display an error state and message for validation.
```tsx
<TextInput
  id="email-input"
  label="Email"
  placeholder="your.email@example.com"
  error="Invalid email format"
  value="invalid@email"
/>
```
### Controlled text input
This example illustrates how to use the TextInput component as a controlled component, managing its value with state.
```tsx
// Assume inputValue is managed by a state hook, e.g.,
// const [inputValue, setInputValue] = useState('');
<TextInput
  id="controlled-input"
  label="Search"
  value={inputValue}
  onChange={(e) => setInputValue(e.target.value)}
  placeholder="Type to search"
/>
