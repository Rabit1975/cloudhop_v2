A flexible image component that supports responsive display and can be easily styled. It ensures images are displayed attractively across different screen sizes.

### Basic image display

This example demonstrates a basic image display using the Image component with default properties.

```tsx
<Image />
```

### Image with responsive sizing
This example shows how to control image dimensions using width and height props, including responsive width.
```tsx
<div style={{ display: 'flex', gap: '32px', alignItems: 'flex-start', flexWrap: 'wrap' }}>
  <div style={{ width: '200px' }}>
    <p style={{ marginBottom: '8px', color: '#888', fontSize: '12px' }}>
      Fixed Width (200px)
    </p>
    <Image src="https://storage.googleapis.com/bit-generated-images/images/image_a_futuristic_and_sleek_digital_0_1770832602892.png" width={200} alt="Fixed width example" />
  </div>
  
  <div style={{ width: '300px' }}>
    <p style={{ marginBottom: '8px', color: '#888', fontSize: '12px' }}>
      Container Width (300px)
    </p>
    <Image src="https://storage.googleapis.com/bit-generated-images/images/image_a_futuristic_and_sleek_digital_0_1770832602892.png" width="100%" alt="Full width example" />
  </div>
  <div>
    <p style={{ marginBottom: '8px', color: '#888', fontSize: '12px' }}>
      Thumbnail (100x100)
    </p>
    <Image 
      src="https://storage.googleapis.com/bit-generated-images/images/image_a_futuristic_and_sleek_digital_0_1770832602892.png" 
      width={100} 
      height={100} 
      style={{ objectFit: 'cover' }} 
      alt="Thumbnail example" 
    />
  </div>
</div>
```
