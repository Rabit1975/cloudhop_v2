A versatile and brand-conscious logo component tailored for CloudHop, with the ability to adapt to diverse presentation needs. It displays the CloudHop logo by default and allows customization via props for name, URL, size, slogan, and optional custom SVG logo.

### Basic Usage

This example demonstrates the basic usage of the Logo component with its default settings.

```tsx
<Logo />
```
### Custom Brand Presentation
This example shows how to customize the logo with a specific name, slogan, and size.
```tsx
<Logo 
  name="CloudHop" 
  slogan="Modular Operating System" 
  size={40} 
/>
```
### Minimal Mode
This example illustrates the minimal mode, where only the logo icon is displayed without any text.
```tsx
<Logo minimal size={32} />
