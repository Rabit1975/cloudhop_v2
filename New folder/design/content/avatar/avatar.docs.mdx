### A basic avatar

This example shows a simple avatar displaying a user image

```tsx
<Avatar src="https://storage.googleapis.com/bit-generated-images/images/image_a_modern__abstract_avatar_icon_0_1770832604911.png" alt="User Avatar" />
```
### Avatar with initials and status indicator
This example demonstrates an avatar displaying initials when no image is provided and showing an online status
```tsx
<Avatar letters="CH" status="online" size="lg" />
```
### Avatar with various sizes and status indicators
This example showcases avatars in different sizes and with various status indicators like online busy away and offline
```tsx
<div>
  <Avatar src="https://storage.googleapis.com/bit-generated-images/images/image_a_modern__abstract_avatar_icon_0_1770832604911.png" size="sm" />
  <Avatar src="https://storage.googleapis.com/bit-generated-images/images/image_a_modern__abstract_avatar_icon_0_1770832604911.png" size="md" status="busy" />
  <Avatar letters="JD" size="lg" status="away" />
  <Avatar letters="AB" size="xl" status="offline" />
</div>
