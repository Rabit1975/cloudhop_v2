### Basic Badge Colors

Displays various badges with different semantic colors for quick status or category identification.

```tsx
import { Badge } from './badge.js';
<div style={{ display: 'flex', gap: '16px', flexWrap: 'wrap' }}>
  <Badge label="Neutral" color="neutral" />
  <Badge label="Primary" color="primary" />
  <Badge label="Positive" color="positive" />
  <Badge label="Warning" color="warning" />
  <Badge label="Error" color="negative" />
  <Badge label="Info" color="info" />
  <Badge label="Secondary" color="secondary" />
</div>
```

### Badge Variants
Illustrates the different visual styles available for the badge component, such as solid, outline, subtle, and ghost.

```tsx

  <div style={{ display: 'flex', gap: '16px', alignItems: 'center' }}>
    <Badge label="Solid" variant="solid" color="primary" />

    <Badge label="Subtle" variant="subtle" color="primary" />

### Badges with Icons and Sizes
Demonstrates how to include an icon within the badge and adjust its size for different contexts.

```tsx
import { Badge } from './badge.js';
function StarIcon() {
  return (
    <svg width="1em" height="1em" viewBox="0 0 24 24" fill="currentColor">
      <path d="M12 2l3.09 6.26L22 9.27l-5 4.87 1.18 6.88L12 17.77l-6.18 3.25L7 14.14 2 9.27l6.91-1.01L12 2z" />
    </svg>
  );
}
function CheckIcon() {
  return (
    <svg width="1em" height="1em" viewBox="0 0 24 24" fill="currentColor">
      <path d="M9 16.17L4.83 12l-1.42 1.41L9 19 21 7l-1.41-1.41z" />
    </svg>
  );
}
<div style={{ display: 'flex', gap: '16px', alignItems: 'center', flexWrap: 'wrap' }}>
  <Badge label="New" icon={<StarIcon />} color="info" size="sm" />
  <Badge label="Success" icon={<CheckIcon />} color="positive" size="md" variant="solid" />
  <Badge label="Featured" icon={<StarIcon />} color="warning" size="lg" variant="subtle" />
</div>
