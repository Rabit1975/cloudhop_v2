### Basic page layout

A fundamental page layout with only primary content and default SEO settings.

```tsx
<PageLayout
  title="My Basic Page"
  description="This is a simple page layout example."
>
  <div>
    <h1>Welcome to My Basic Page</h1>
    <p>This is the main content area of the page.</p>
  </div>
</PageLayout>
```
### Page layout with navigation tabs
An example demonstrating a page layout including internal navigation tabs and custom content.
```tsx
const tabs = [
  { label: 'Overview', href: '/overview' },
  { label: 'Settings', href: '/settings' },
  { label: 'Activity', href: '/activity' },
];
<PageLayout
  title="User Profile"
  description="Manage your user profile and settings"
  tabs={tabs}
>
  <div>
    <h2>Profile Overview</h2>
    <p>View and edit your personal information here.</p>
  </div>
</PageLayout>
```
### Page layout with custom styling
A page layout applying additional custom styling to its content area.
```tsx
<PageLayout
  title="Custom Styled Content"
  description="A page demonstrating custom styling for the content area"
  contentClassName="my-custom-content-background" // This class would be defined in your global CSS or a parent component
>
  <div style={{ padding: '20px', backgroundColor: '#f0f0f0', borderRadius: '8px' }}>
    <h3>Content with a custom background</h3>
    <p>This area has specific styling applied via contentClassName.</p>
  </div>
</PageLayout>
