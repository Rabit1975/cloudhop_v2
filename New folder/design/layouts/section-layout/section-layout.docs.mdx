A visually striking adaptable page section component that elevates content presentation. It allows for flexible content arrangement, with optional title, caption, and subtitle elements for clear communication.

## Basic usage with title, subtitle, and caption

Render a standard section with a caption, title, and subtitle, and left aligned content.

```tsx
<SectionLayout
  caption="HopHub Features"
  title="Connect with your Community"
  subtitle="Real-time group chat, channels, and direct messaging all in one place. Experience the next generation of social interaction."
>
  <div style={{
    padding: '24px',
    backgroundColor: 'var(--colors-surface-primary)',
    borderRadius: 'var(--borders-radius-medium)',
    border: '1px solid var(--colors-border-subtle)',
    color: 'var(--colors-text-primary)'
  }}>
    <h3>Your Main Content Here</h3>
    <p style={{ color: 'var(--colors-text-secondary)', marginTop: '8px' }}>
      This is where your primary section content will be displayed.
    </p>
  </div>
</SectionLayout>
```
## Center aligned content with an image
Demonstrate a section with all text elements and content centered, featuring an image as its main content.
```tsx
<SectionLayout
  align="center"
  caption="Unified Experience"
  title="Experience the Nebula"
  subtitle="The nebula background creates visual distinction for the social hub while keeping other sections focused and distraction-free."
>
  <img
    src="https://storage.googleapis.com/bit-generated-images/images/image_a_futuristic__minimalist_abstr_0_1770833866348.png"
    alt="Futuristic Abstract"
    style={{
      width: '100%',
      maxWidth: '600px',
      height: 'auto',
      display: 'block',
      margin: '0 auto',
      borderRadius: 'var(--borders-radius-large)',
      overflow: 'hidden',
      boxShadow: 'var(--effects-shadows-large)',
      marginTop: '16px'
    }}
  />
</SectionLayout>
```
