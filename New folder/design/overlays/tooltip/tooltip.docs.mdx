A tooltip component displays supplementary information when a user hovers over or focuses on an interactive element. It provides contextual details without cluttering the main interface.

### Basic Tooltip Usage

This example demonstrates a basic tooltip showing a simple text message on hover over a button.

```tsx
<Tooltip content="Add to favorites">
  <button style={{
    padding: '10px 20px',
    backgroundColor: '#007bff',
    color: '#fff',
    border: 'none',
    borderRadius: '5px',
    cursor: 'pointer',
    fontSize: '16px'
  }}>
    Hover Me
  </button>
</Tooltip>
```
