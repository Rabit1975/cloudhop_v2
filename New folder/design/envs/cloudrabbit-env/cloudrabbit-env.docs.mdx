### Applying the environment to a component

This example shows how to set the Cloudrabbit environment on an existing Bit component.

```bash
bit env set my-component cloudrabbit.env/cloudrabbit-env
```
### Creating a new workspace with Cloudrabbit environment
This example demonstrates how to initialize a new Bit workspace using the Cloudrabbit environment as the default.
```bash
bit new react my-cloudhop-workspace --aspect cloudrabbit.env/cloudrabbit-env
```
### Extending the Cloudrabbit environment
This example illustrates how to create a new custom environment that extends the Cloudrabbit environment, allowing for further customization.
```bash
bit create react-env my-custom-cloudrabbit-env --aspect cloudrabbit.env/cloudrabbit-env
