---

A component to embed and play HTML5 or Unity WebGL games within your application. It provides a consistent interface for launching various game types, optionally displaying a cover image and title before the game starts.

### Embedding an HTML5 Game

This example demonstrates embedding an HTML5 game with a cover image. The game will start playing when the user clicks the play button.

```tsx
<GamePlayer
  title="Cosmic Explorer Adventure"
  platform="html5"
  gameUrl="https://example.com/html5-game/index.html"
  coverUrl="https://storage.googleapis.com/bit-generated-images/images/image_a_dynamic__close_up_shot_of_a__0_1770832617236.png"
/>
```
### Autoplay for Unity Game
This example shows how to embed a Unity WebGL game that begins playing automatically without requiring a click on a play button.
```tsx
<GamePlayer
  title="Nebula Racer 3D"
  platform="unity"
  gameUrl="https://example.com/unity-game/index.html"
  autoPlay
/>
```
