A flexible component to display information about a game, including its thumbnail, title, genre, platform, and rating. It supports custom headers and footers and is clickable to navigate to game details.

### Basic Game Card

A simple game card displaying essential game information, assuming the game object is provided.

```tsx
const basicGame = {
  id: 'game-1',
  name: 'Cyber Racer X',
  imageUrls: ['https://storage.googleapis.com/bit-generated-images/images/image_high_resolution_game_thumbnail_0_1770833871150.png'],
  gameType: 'html5',
  tags: ['Racing', 'Cyberpunk'],
};
<div style={{ maxWidth: 300, padding: 20 }}>
  <GameCard game={basicGame} />
</div>
```

### Game Card with Rating and Custom Footer
This example shows a game card with a rating and custom content in the footer section.
```tsx
const ratedGame = {
  id: 'game-2',
  name: 'Nebula Quest',
  imageUrls: ['https://storage.googleapis.com/bit-generated-images/images/image_high_resolution_game_thumbnail_0_1770833871150.png'],
  gameType: 'unity',
  tags: ['RPG', 'Adventure'],
};
<div style={{ maxWidth: 300, padding: 20 }}>
  <GameCard 
    game={ratedGame} 
    rating={4.8}
    footer={
      <div style={{ fontSize: '12px', color: 'var(--colors-text-secondary)' }}>
        Played by 12k users
      </div>
    }
  />
</div>
```

### Game Card with Custom Header
Demonstrates how to add custom content to the header of the game card.
```tsx
const streamedGame = {
  id: 'game-3',
  name: 'Pro Tournament Live',
  imageUrls: ['https://storage.googleapis.com/bit-generated-images/images/image_high_resolution_game_thumbnail_0_1770833871150.png'],
  gameType: 'external_stream',
  tags: ['Esports', 'FPS'],
};
<div style={{ maxWidth: 300, padding: 20 }}>
  <GameCard 
    game={streamedGame} 
    header={
      <div style={{ 
        display: 'flex', 
        justifyContent: 'space-between', 
        alignItems: 'center', 
        width: '100%' 
      }}>
        <span style={{ 
          color: 'var(--colors-status-negative-default)', 
          fontSize: '10px', 
          fontWeight: 'bold', 
          display: 'flex',
          alignItems: 'center',
          gap: '4px'
        }}>
          <span style={{ width: 6, height: 6, borderRadius: '50%', background: 'currentColor' }} />
          LIVE
        </span>
        <span style={{ fontSize: '10px', color: 'var(--colors-text-secondary)' }}>
          Twitch
        </span>
      </div>
    }
  />
</div>
