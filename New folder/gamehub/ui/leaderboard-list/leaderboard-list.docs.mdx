A component for displaying game leaderboards, showing player scores, usernames, and timestamps. It can highlight the current user's entry.

## Component usage

### Basic Leaderboard Display

This example demonstrates how to render a simple leaderboard with player scores.

```tsx
const leaderboard = {
  gameId: 'daily-challenge',
  scores: [
    { userId: 'user-101', score: 5000, timestamp: 1678886400000 },
    { userId: 'user-102', score: 4500, timestamp: 1678879200000 },
    { userId: 'user-103', score: 3800, timestamp: 1678792800000 },
  ],
};
const users = [
  { userId: 'user-101', username: 'SwiftPaws', displayName: 'Swift Paws' },
  { userId: 'user-102', username: 'Zenith', displayName: 'Zenith' },
  { userId: 'user-103', username: 'NightHawk', displayName: 'Night Hawk' },
];
<LeaderboardList
  leaderboard={leaderboard}
  users={users}
/>
```
### Highlighting a Current User Score
This example shows how to highlight the score of a specific user within the leaderboard.
```tsx
const leaderboard = {
  gameId: 'weekly-cup',
  scores: [
    { userId: 'player-A', score: 7500, timestamp: 1678900000000 },
    { userId: 'player-B', score: 6800, timestamp: 1678896400000 },
    { userId: 'player-C', score: 6200, timestamp: 1678903600000 },
    { userId: 'player-D', score: 5900, timestamp: 1678892800000 },
  ],
};
const users = [
  { userId: 'player-A', username: 'EpicGamer', displayName: 'Epic Gamer' },
  { userId: 'player-B', username: 'PixelKing', displayName: 'Pixel King' },
  { userId: 'player-C', username: 'CurrentPlayer', displayName: 'You' },
  { userId: 'player-D', username: 'Speedster', displayName: 'Speedster' },
];
<LeaderboardList
  leaderboard={leaderboard}
  users={users}
  currentUserId="player-C"
/>
```
### Loading and Empty States
This example illustrates the loading indicator and the message shown when the leaderboard has no scores.
```tsx
const sampleLeaderboard = {
  gameId: 'sample-game',
  scores: [
    { userId: 'user-X', score: 9000, timestamp: 1678900000000 },
  ],
};
const sampleUsers = [
  { userId: 'user-X', username: 'Tester', displayName: 'The Tester' },
];
const emptyLeaderboard = {
  gameId: 'empty-game',
  scores: [],
};
<div>
  <h4>Loading State</h4>
  <LeaderboardList
    leaderboard={sampleLeaderboard}
    users={sampleUsers}
    loading={true}
  />
</div>
<div style={{ marginTop: '30px' }}>
  <h4>Empty State</h4>
  <LeaderboardList
    leaderboard={emptyLeaderboard}
    users={[]}
  />
</div>
