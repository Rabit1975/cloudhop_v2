### Basic Activity Feed

Displays a standard activity feed with various types of game activities.

```tsx
const mockGames = [
  {
    id: 'game-1',
    name: 'Nebula Racers',
  },
  {
    id: 'game-2',
    name: 'Cosmic Chess',
  },
  {
    id: 'game-3',
    name: 'Star Streamer',
  },
];
const mockActivities = [
  {
    id: 'act-1',
    userId: 'user-1',
    gameId: 'game-1',
    type: 'played',
    timestamp: new Date(Date.now() - 1000 * 60 * 60 * 2).toISOString(),
    details: { score: 15400 },
  },
  {
    id: 'act-2',
    userId: 'user-1',
    gameId: 'game-2',
    type: 'achieved',
    timestamp: new Date(Date.now() - 1000 * 60 * 60 * 24).toISOString(),
    details: { achievementId: 'ach-1' },
  },
  {
    id: 'act-3',
    userId: 'user-1',
    gameId: 'game-3',
    type: 'streamed',
    timestamp: new Date(Date.now() - 1000 * 60 * 60 * 48).toISOString(),
    details: { streamId: 'stream-1' },
  },
];
<GameActivityFeed activities={mockActivities} games={mockGames} />
```

### Empty Activity Feed

Demonstrates the appearance of the feed when there are no activities to display.

```tsx
<GameActivityFeed activities={[]} />
```

### Activity Feed with Custom Empty State

Illustrates how to provide a custom message when there are no activities.

```tsx
<GameActivityFeed
  activities={[]}
  emptyState={<div>No gaming adventures to show yet. Go play some games!</div>}
/>
