### Basic Game Icon

Renders the default game controller icon.

```tsx
<GameIcon />
```

### Game Icon Sizing
Adjust the icon size using the size prop.
```tsx
<div style={{ display: 'flex', gap: '16px', alignItems: 'center' }}>
  <GameIcon size={16} />
  <GameIcon size={24} />
  <GameIcon size={32} />
  <GameIcon size={48} />
</div>
```

### Game Icon Coloring
Change the icon color using the color prop.
```tsx
<div style={{ display: 'flex', gap: '16px', alignItems: 'center' }}>
  <GameIcon color="red" />
  <GameIcon color="blue" />
  <GameIcon color="green" />
  <GameIcon color="#800080" />
</div>
```
