A component for a CalendarEvent entity, supporting serialization and deserialization of CalendarEvent objects, and usable across both frontend and backend systems.

## Get started

### Creating a basic event
This example demonstrates how to create a new calendar event instance with essential details.

```ts
import { CalendarEvent } from '@cloudhop/calendar-event';
const now = new Date();
const tomorrow = new Date(now.getTime() + 24 * 60 * 60 * 1000);
const event = CalendarEvent.from({
  id: 'event-001',
  title: 'Team Stand-up',
  description: 'Daily team synchronization meeting.',
  startTime: now.toISOString(),
  endTime: tomorrow.toISOString(),
  attendees: ['john.doe@example.com', 'jane.smith@example.com'],
  location: 'Virtual Meeting Room',
  meetingId: 'meet-12345'
});
console.log(event.title);
// => Team Stand-up
```
### Converting between plain objects and CalendarEvent instances
This example shows how to convert a CalendarEvent instance to a plain object and back using the static from method.
```ts
import { CalendarEvent } from '@cloudhop/calendar-event';
const originalEvent = CalendarEvent.from({
  id: 'event-002',
  title: 'Project Kick-off',
  startTime: '2023-11-15T09:00:00Z',
  endTime: '2023-11-15T10:00:00Z',
  attendees: ['alice', 'bob'],
  location: 'Main Conference Room'
});
const plainObject = originalEvent.toObject();
console.log(plainObject.location);
// => Main Conference Room
const recreatedEvent = CalendarEvent.from(plainObject);
console.log(recreatedEvent.attendees);
// => ['alice', 'bob']
```
