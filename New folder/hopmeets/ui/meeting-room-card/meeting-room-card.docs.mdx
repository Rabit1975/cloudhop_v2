The Meeting Room Card component provides a compact and informative display for a single meeting room. It shows key details like the room name, capacity, privacy status, and whether a meeting is currently in progress. Users can quickly enter a room or view its details.

### Basic Meeting Room Card

This example displays a standard meeting room card, showing basic information and actions.

```tsx
<MeetingRoomCard
  room={{
    id: 'room-01',
    name: 'Open Lounge',
    capacity: 50,
    isPrivate: false,
    ownerId: 'system',
    currentMeetingId: undefined,
  }}
  onEnter={(roomId) => console.log(`Entering room: ${roomId}`)}
  detailsHref="/hopmeets/room-01"
/>
```
### Private Meeting Room Card
This example demonstrates a meeting room marked as private, indicating it may require specific access.
```tsx
<MeetingRoomCard
  room={{
    id: 'room-03',
    name: 'Executive Board Room',
    capacity: 8,
    isPrivate: true,
    ownerId: 'admin',
    currentMeetingId: undefined,
  }}
  onEnter={(roomId) => console.log(`Entering private room: ${roomId}`)}
  detailsHref="/hopmeets/room-03"
/>
```
### Live Meeting Room Card
This example shows a meeting room that is currently active with an ongoing meeting.
```tsx
<MeetingRoomCard
  room={{
    id: 'room-02',
    name: 'Daily Standup',
    capacity: 15,
    isPrivate: false,
    ownerId: 'team-lead',
    currentMeetingId: 'meeting-active-123',
  }}
  onEnter={(roomId) => console.log(`Joining live meeting in room: ${roomId}`)}
  detailsHref="/hopmeets/room-02"
/>
