A video call icon component used to represent video conferencing functionality.

### Basic VideoCall Icon

This shows the default rendering of the VideoCall icon

```jsx
<VideoCall />
```
### VideoCall Icon with Different Sizes
This example demonstrates how to render the VideoCall icon in various sizes
```jsx
<div>
  <VideoCall size={16} />
  <VideoCall size={24} />
  <VideoCall size={32} />
  <VideoCall size={48} />
</div>
```
### VideoCall Icon in a Button
This illustrates using the VideoCall icon as part of an interactive element like a button
