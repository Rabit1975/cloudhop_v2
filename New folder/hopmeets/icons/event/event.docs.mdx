### Basic Event Icon

Render the Event icon with its default size and color.

```tsx
<Event />
```
### Event Icon Sizes
Adjust the visual size of the Event icon using the size prop to fit various layout needs.
```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: '24px' }}>
  <Event size={16} />
  <Event size={24} />
  <Event size={32} />
  <Event size={48} />
</div>
```
### Event Icon Colors
Change the fill color of the Event icon using the color prop.
```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: '24px' }}>
  <Event color="blue" />
  <Event color="green" />
  <Event color="purple" />
  <Event color="red" />
</div>
