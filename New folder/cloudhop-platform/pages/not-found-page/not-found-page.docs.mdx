### Basic rendering of the 404 page

This shows the default rendering of the 404 not found page for CloudHop.

```jsx
<NotFoundPage />
```
### Customizing the appearance with a class name
You can apply a custom class name to the root element of the 404 page for additional styling or layout adjustments.
```jsx
<NotFoundPage className="my-custom-404-style" />
```
### Rendering the page within a restricted space
The 404 page can adapt to different container sizes maintaining its layout and responsiveness. This example shows it in a height-limited container.
```jsx
<div style={{ height: '500px', border: '1px solid var(--colors-border-default)', overflow: 'hidden' }}>
  <NotFoundPage style={{ minHeight: '100%', padding: 'var(--spacing-medium)' }} />
</div>
