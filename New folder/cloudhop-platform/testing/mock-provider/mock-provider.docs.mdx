A MockProvider for testing components across the CloudHop platform. Use it to render components with mock data, test components, or render with mock data.

## Basic usage

Render components inside the MockProvider to provide a mocked environment. This example also shows how to use the `useIsMock` hook to check if the component is running in a mocked context.

```js
const MockStatus = () => {
  const isMock = useIsMock();
  
  return (
    <div style={{ 
      padding: '24px', 
      borderRadius: 'var(--borders-radius-medium, 12px)',
      backgroundColor: 'var(--colors-surface-primary, #2a2a2a)',
      border: '1px solid var(--colors-border-default, #444)',
      boxShadow: 'var(--effects-shadows-medium)'
    }}>
      <h3 style={{ 
        marginTop: 0, 
        color: 'var(--colors-text-primary, #fff)',
        fontSize: 'var(--typography-sizes-heading-h4)'
      }}>
        Environment Status
      </h3>
      <div style={{ display: 'flex', gap: '8px', alignItems: 'center' }}>
        <span style={{ color: 'var(--colors-text-secondary, #ccc)' }}>Is Mock Mode:</span>
        <span style={{ 
          fontWeight: 'bold', 
          color: isMock ? 'var(--colors-positive-text, #4caf50)' : 'var(--colors-negative-text)'
        }}>
          {isMock ? 'Active' : 'Inactive'}
        </span>
      </div>
    </div>
  );
};
export const BasicMockProvider = () => {
  return (
    <MockProvider>
      <div style={{ padding: '48px' }}>
        <h1 style={{ 
          color: 'var(--colors-text-primary',
          marginBottom: '24px',
          background: 'var(--effects-gradients-primary)',
          WebkitBackgroundClip: 'text',
          WebkitTextFillColor: 'transparent',
          display: 'inline-block'
        }}>
          CloudHop Provider
        </h1>
        <MockStatus />
      </div>
    </MockProvider>
  );
};
```

## Using with React Router

The MockProvider automatically includes a MemoryRouter for easy testing of components that rely on routing. This allows you to simulate different routes within your component tests.

```js
const RoutingDemo = () => {
  const location = useLocation();
  return (
    <div style={{ padding: '32px', color: 'var(--colors-text-primary)' }}>
      <div style={{ marginBottom: '24px' }}>
        <div style={{ 
          fontSize: '12px', 
          textTransform: 'uppercase', 
          letterSpacing: '1px',
          color: 'var(--colors-text-secondary)',
          marginBottom: '8px'
        }}>
          Current Route
        </div>
        <div style={{ 
          fontFamily: 'monospace', 
          padding: '8px 12px', 
          backgroundColor: 'var(--colors-surface-secondary)', 
          borderRadius: '6px',
          display: 'inline-block'
        }}>
          {location.pathname}
        </div>
      </div>
      <div style={{ display: 'flex', gap: '16px', marginBottom: '32px' }}>
        {['/', '/chat', '/settings'].map((path) => (
          <Link
            key={path}
            to={path}
            style={{
              padding: '8px 16px',
              borderRadius: '6px',
              textDecoration: 'none',
              backgroundColor: location.pathname === path 
                ? 'var(--colors-primary-default)' 
                : 'var(--colors-surface-secondary)',
              color: location.pathname === path 
                ? 'var(--colors-primary-contrast)' 
                : 'var(--colors-text-primary)',
              transition: 'all 0.2s ease'
            }}
          >
            {path === '/' ? 'Overview' : path.slice(1).charAt(0).toUpperCase() + path.slice(2)}
          </Link>
        ))}
      </div>
      <div style={{ 
        padding: '24px', 
        backgroundColor: 'var(--colors-surface-primary)', 
        borderRadius: '12px',
        border: '1px solid var(--colors-border-subtle)'
      }}>
        <Routes>
          <Route path="/" element={<div>Select a module to begin.</div>} />
          <Route path="/chat" element={<div>Chat Module Loaded</div>} />
          <Route path="/settings" element={<div>Settings Module Loaded</div>} />
        </Routes>
      </div>
    </div>
  );
};
export const WithRouting = () => {
  return (
    <MockProvider>
      <RoutingDemo />
    </MockProvider>
  );
};
```

## Disabling the theme

By default, the MockProvider wraps components with the Cloudrabbit theme. You can disable this behavior if your component does not require theme-specific styling.

```js
export const NoTheme = () => {
  return (
    <div style={{ padding: '20px', border: '1px solid #ccc' }}>
      <p style={{ marginBottom: '16px' }}>Theme disabled (native styles):</p>
      <MockProvider noTheme>
        <button>Native Button</button>
      </MockProvider>
    </div>
  );
};
