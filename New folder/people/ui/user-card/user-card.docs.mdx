A user card component optimized for lists and grids, emphasizing clear visual hierarchy. It includes a large, interactive Avatar, a dynamically sized user name, and an optional, subtle status message.

## Basic User Card

Displays a single user card with an avatar, name, and status.

```tsx
<BasicUserCard />
```
## User Cards in a Grid
Illustrates multiple user cards arranged in a grid, showcasing various presence statuses.
```tsx
<UserGrid />
```
## User Card Visual Variants
Demonstrates the different visual styles available for the user card, such as elevated, outlined, and default.
