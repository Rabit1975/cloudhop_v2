A component for a Friend Request entity, supports serialization and de-serialization of Friend Request objects and can be used from both backend and frontend.

## Create a Friend Request

Learn how to create a friend request instance from a plain object.

```ts
import { FriendRequest } from './friend-request.js';
const newRequest = FriendRequest.from({
  id: 'req_new_1',
  fromUserId: 'user_alice',
  toUserId: 'user_bob',
  status: 'pending',
  createdAt: new Date().toISOString(),
});
console.log(newRequest.id);
console.log(newRequest.status);
```

## Access Friend Request Properties
Understand how to access properties like senderId, receiverId, and sentAt, and how to convert the object back to a plain object.

```ts
import { FriendRequest } from './friend-request.js';
const acceptedRequest = FriendRequest.from({
  id: 'req_002',
  fromUserId: 'user_charlie',
  toUserId: 'user_diana',
  status: 'accepted',
  createdAt: '2023-11-01T10:00:00Z',
});
console.log("Sender ID", acceptedRequest.senderId);
console.log("Receiver ID", acceptedRequest.receiverId);
console.log("Sent At", acceptedRequest.sentAt);
const plainObject = acceptedRequest.toObject();
console.log(plainObject);
```

// Get a list of predefined mock requests
const allMockRequests = mockFriendRequests;
