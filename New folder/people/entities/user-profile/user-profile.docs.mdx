A component for a UserProfile entity, supports serialization and de-serialization of UserProfile objects and can be used from
both backend and frontend.

## Get started

Install and use the component in your code:

```ts
import { UserProfile } from '@cloudrabbit/people.entities.user-profile';
import type { PlainSocialConnection } from '@cloudrabbit/people.entities.social-connection';
import type { UserPresenceStatus } from '@cloudrabbit/people.entities.presence-status';
const plainUserProfile = {
  id: 'profile-xyz',
  userId: 'user-abc',
  firstName: 'Alice',
  lastName: 'Smith',
  email: 'alice.smith@example.com',
  profilePicture: 'https://example.com/alice.jpg',
  statusMessage: 'Hello CloudHop!',
  presenceStatus: 'online',
  socialConnections: [
    {
      platform: 'Twitter',
      username: 'alicesmith',
      url: 'https://twitter.com/alicesmith'
    }
  ],
  createdAt: new Date().toISOString(),
};
const userProfile = UserProfile.from(plainUserProfile);
console.log(userProfile.firstName);
```

### Serialize a UserProfile

This example shows how to convert a UserProfile instance back into a plain object, useful for data transfer or storage.

```ts
import { UserProfile } from '@cloudrabbit/people.entities.user-profile';
import type { PlainSocialConnection } from '@cloudrabbit/people.entities.social-connection';
import type { UserPresenceStatus } from '@cloudrabbit/people.entities.presence-status';
const plainUserProfile = {
  id: 'profile-xyz',
  userId: 'user-abc',
  firstName: 'Alice',
  lastName: 'Smith',
  email: 'alice.smith@example.com',
  profilePicture: 'https://example.com/alice.jpg',
  statusMessage: 'Hello CloudHop!',
  presenceStatus: 'online',
  socialConnections: [
    {
      platform: 'Twitter',
      username: 'alicesmith',
      url: 'https://twitter.com/alicesmith'
    }
  ],
  createdAt: new Date().toISOString(),
};
const userProfile = UserProfile.from(plainUserProfile);
const serializedProfile = userProfile.toObject();
console.log(serializedProfile.userId);
```

### Use mock UserProfile data

This example shows how to use the provided mock data utility for development and testing purposes.

```ts
import { mockUserProfile, mockUserProfiles } from '@cloudrabbit/people.entities.user-profile';
const singleProfile = mockUserProfile({ firstName: 'Test', lastName: 'User' });
console.log(singleProfile.firstName);
const multipleProfiles = mockUserProfiles();
console.log(`Found ${multipleProfiles.length} mock profiles.`);
