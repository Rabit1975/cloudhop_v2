A component that renders a standard user icon, providing a consistent visual representation for user profiles or avatars across the application. It leverages a base icon component for scalable sizing and styling.

### Default User Icon

Displays the user icon with its default size and color.

```jsx
<UserIcon />
```

### User Icon with Custom Sizes
Demonstrates how to adjust the size of the user icon using the size property.
```jsx
<div>
  <UserIcon size={16} />
  <UserIcon size={24} />
  <UserIcon size={32} />
  <UserIcon size={48} />
</div>
```

### User Icon with Custom Colors
Shows how to change the fill color of the user icon using the color property.
```jsx
<div>
  <UserIcon color="blue" />
  <UserIcon color="green" />
  <UserIcon color="red" />
</div>
```

### User Icon in a Profile Context
An example of using the user icon within a simple layout, useful for profile previews or avatars.
```jsx
<div style={{ display: 'flex', alignItems: 'center', gap: '8px', padding: '16px', border: '1px solid #eee', borderRadius: '8px' }}>
  <div style={{ width: '32px', height: '32px', borderRadius: '50%', backgroundColor: '#f0f0f0', display: 'flex', alignItems: 'center', justifyContent: 'center' }}>
    <UserIcon size={20} color="#666" />
  </div>
  <span style={{ fontWeight: 'bold', fontSize: '1rem' }}>John Doe</span>
</div>
