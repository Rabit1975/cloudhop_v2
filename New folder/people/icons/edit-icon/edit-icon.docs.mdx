An edit icon component, useful for indicating editable elements or actions.

### Basic usage

Renders the default edit icon.

```tsx
<EditIcon />
```
### Custom sizes
Customize the size of the edit icon using the size prop.
```tsx
<div style={{ display: 'flex', gap: '16px', alignItems: 'center' }}>
  <EditIcon size={16} />
  <EditIcon size={24} />
  <EditIcon size={32} />
  <EditIcon size={48} />
</div>
```
### Custom colors
Change the color of the edit icon using the color prop.
```tsx
<div style={{ display: 'flex', gap: '16px', alignItems: 'center' }}>
  <EditIcon color="blue" />
  <EditIcon color="green" />
  <EditIcon color="red" />
</div>
