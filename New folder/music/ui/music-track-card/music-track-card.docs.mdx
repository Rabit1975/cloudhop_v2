A Music Track Card visually represents a music track, displaying its thumbnail, title, and artist. It includes actions for playing the track and adding it to a playlist. The entire card is clickable to navigate to a track detail or playback page.

## Component usage

```tsx
import { MusicTrackCard } from './music-track-card.js';
// Assuming you have a MusicTrackPlain type or similar structure
const sampleTrack = {
  id: '1',
  title: 'CloudHop Anthem',
  artist: 'The CloudHoppers',
  thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_a_vibrant__modern_music_album__0_1770835136296.png',
  duration: 240, // in seconds
  album: 'CloudHop Hits',
  releaseDate: '2023-01-01T00:00:00Z',
  genre: 'Electronic',
};
return (
  <MusicTrackCard
    track={sampleTrack}
    onPlay={(trackId) => console.log(`Playing track: ${trackId}`)}
    onAddToPlaylist={(trackId) => console.log(`Adding track to playlist: ${trackId}`)}
    href="/music/track/1"
  />
);
```

### Basic Music Track Card

This example demonstrates a basic music track card showing track details and action buttons

```tsx
import { MusicTrackCard } from './music-track-card.js';
const track1 = {
  id: 'track-001',
  title: 'Nebula Serenade',
  artist: 'Aura Sync',
  thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_a_vibrant__modern_music_album__0_1770835136296.png',
  duration: 210, // 3:30
  album: 'Cosmic Harmonies',
  releaseDate: '2023-05-10T00:00:00Z',
  genre: 'Ambient',
};
return (
  <div style={{ maxWidth: '300px', margin: '20px auto' }}>
    <MusicTrackCard
      track={track1}
      onPlay={(id) => alert(`Playing: ${id}`)}
      onAddToPlaylist={(id) => alert(`Added to playlist: ${id}`)}
    />
  </div>
);
```

### Navigable Music Track Card

This example shows a music track card that acts as a link, navigating to a specified URL when clicked

```tsx
import { MusicTrackCard } from './music-track-card.js';
const track2 = {
  id: 'track-002',
  title: 'Digital Echoes',
  artist: 'Byte Beat',
  thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_a_vibrant__modern_music_album__0_1770835136296.png',
  duration: 185, // 3:05
  album: 'Cybernetic Dreams',
  releaseDate: '2023-07-22T00:00:00Z',
  genre: 'Synthwave',
};
return (
  <div style={{ maxWidth: '300px', margin: '20px auto' }}>
    <MusicTrackCard
      track={track2}
      href="/music/track-detail/digital-echoes"
      onPlay={(id) => console.log(`Playing: ${id}`)}
      onAddToPlaylist={(id) => console.log(`Adding to playlist: ${id}`)}
    />
  </div>
);
```
