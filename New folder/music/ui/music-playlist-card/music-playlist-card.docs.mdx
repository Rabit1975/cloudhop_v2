### Basic Playlist Card

This example displays a public playlist with a cover image, its name, and the number of tracks. Clicking the card navigates to the playlist detail page. The play button triggers a callback.

```tsx
const mockPublicPlaylist = {
  id: 'playlist-1',
  name: 'Nebula Vibes',
  description: 'Chill beats for coding and space travel.',
  isPublic: true,
  ownerId: 'user-1',
  createdAt: '2023-01-01T12:00:00Z',
  tracks: [
    {
      id: 't1',
      title: 'Starlight',
      duration: 180,
      sourceType: 'upload',
      sourceIdentifier: 'file-1',
      isPublic: true,
      createdAt: '2023-01-01T12:00:00Z',
      thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_music_playlis_0_1770835233075.png'
    },
    {
      id: 't2',
      title: 'Cosmic Drift',
      duration: 240,
      sourceType: 'upload',
      sourceIdentifier: 'file-2',
      isPublic: true,
      createdAt: '2023-01-02T12:00:00Z',
      thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_music_playlis_0_1770835233075.png'
    },
  ],
};
<div style={{ padding: 32, maxWidth: 300 }}>
  <MusicPlaylistCard 
    playlist={mockPublicPlaylist} 
    onPlay={(id) => alert(`Playing playlist: ${id}`)} 
  />
</div>
```
### Private Playlist Card
This example shows a playlist marked as private. It features a different badge to indicate its status and has no tracks, which adjusts the track count display.
```tsx
const mockPrivatePlaylist = {
  id: 'playlist-2',
  name: 'Top Secret Jams',
  description: 'Only for the privileged ears.',
  isPublic: false,
  ownerId: 'user-1',
  createdAt: '2023-01-01T12:00:00Z',
  tracks: [],
};
<div style={{ padding: 32, maxWidth: 300 }}>
  <MusicPlaylistCard 
    playlist={mockPrivatePlaylist} 
    onPlay={(id) => alert(`Playing playlist: ${id}`)} 
  />
</div>
```
### Playlist Cards in a Grid
This example demonstrates how multiple playlist cards can be arranged in a responsive grid layout. It showcases different playlist states, including one with no description and another with many tracks.
```tsx
const mockPublicPlaylist = {
  id: 'playlist-1',
  name: 'Nebula Vibes',
  description: 'Chill beats for coding and space travel.',
  isPublic: true,
  ownerId: 'user-1',
  createdAt: '2023-01-01T12:00:00Z',
  tracks: [
    {
      id: 't1',
      title: 'Starlight',
      duration: 180,
      sourceType: 'upload',
      sourceIdentifier: 'file-1',
      isPublic: true,
      createdAt: '2023-01-01T12:00:00Z',
      thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_music_playlis_0_1770835233075.png'
    },
    {
      id: 't2',
      title: 'Cosmic Drift',
      duration: 240,
      sourceType: 'upload',
      sourceIdentifier: 'file-2',
      isPublic: true,
      createdAt: '2023-01-02T12:00:00Z',
      thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_music_playlis_0_1770835233075.png'
    },
  ],
};
const mockPrivatePlaylist = {
  id: 'playlist-2',
  name: 'Top Secret Jams',
  description: 'Only for the privileged ears.',
  isPublic: false,
  ownerId: 'user-1',
  createdAt: '2023-01-01T12:00:00Z',
  tracks: [],
};
const mockGymPlaylist = {
  ...mockPublicPlaylist,
  id: 'playlist-3',
  name: 'Gym Motivation',
  description: undefined,
};
const mockSleepTrack = {
  id: 't-sleep',
  title: 'Ambient Sound',
  duration: 300,
  sourceType: 'upload',
  sourceIdentifier: 'file-sleep',
  isPublic: true,
  createdAt: '2023-01-01T12:00:00Z',
  thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_music_playlis_0_1770835233075.png'
};
const mockSleepPlaylist = {
  ...mockPublicPlaylist,
  id: 'playlist-4',
  name: 'Sleep & Relax',
  tracks: Array(50).fill(mockSleepTrack),
};
<div style={{ 
  padding: 32, 
  display: 'grid', 
  gridTemplateColumns: 'repeat(auto-fill, minmax(240px, 1fr))', 
  gap: 24,
  backgroundColor: 'var(--colors-surface-background)'
}}>
  <MusicPlaylistCard playlist={mockPublicPlaylist} />
  <MusicPlaylistCard playlist={mockPrivatePlaylist} />
  <MusicPlaylistCard playlist={mockGymPlaylist} />
  <MusicPlaylistCard playlist={mockSleepPlaylist} />
</div>
