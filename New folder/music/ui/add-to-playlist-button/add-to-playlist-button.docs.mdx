A UI component that provides functionality to add a given music track to a user's existing playlist or to a newly created playlist.

## Component usage

The AddToPlaylistButton allows users to manage their music collections by easily organizing tracks into custom playlists.

```tsx
import { MusicTrack } from '@cloudrabbit/music.entities.music-track';

const track = new MusicTrack(
  'track-1',
  'Stellar Drifting',
  245,
  'youtube',
  'abc12345',
  true,
  new Date().toISOString(),
  'Cosmic Beats',
  'https://example.com/thumbnail.jpg'
);

<AddToPlaylistButton track={track} />
```

### Basic component usage
Render the add to playlist button with a music track
```tsx
import { MusicTrack } from '@cloudrabbit/music.entities.music-track';

const mockTrack = new MusicTrack(
  'track-1',
  'Stellar Drifting',
  245,
  'youtube',
  'abc12345',
  true,
  new Date().toISOString(),
  'Cosmic Beats',
  'https://example.com/thumbnail.jpg'
);
<AddToPlaylistButton track={mockTrack} />
```
