A simple music note icon for use in various UI contexts, representing music-related features or content.

### Basic Usage

A basic render of the music note icon.

```tsx
<MusicNoteIcon />
```
### Different Sizes
The music note icon can be rendered in various sizes using the size property.
```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: '24px' }}>
  <MusicNoteIcon size={16} />
  <MusicNoteIcon size={24} />
  <MusicNoteIcon size={32} />
  <MusicNoteIcon size={48} />
</div>
```
### Different Colors
The music note icon can be customized with different colors.
```tsx
<div style={{ display: 'flex', alignItems: 'center', gap: '24px' }}>
  <MusicNoteIcon color="#FF0000" />
  <MusicNoteIcon color="#00FF00" />
  <MusicNoteIcon color="#0000FF" />
</div>
