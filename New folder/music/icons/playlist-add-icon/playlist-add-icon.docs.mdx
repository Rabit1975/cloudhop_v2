An icon component representing the action of adding an item to a playlist.

### Basic Playlist Add Icon

This shows the default rendering of the PlaylistAddIcon component.

```tsx
<PlaylistAddIcon />
```

### Icon with Different Sizes

The icon can be rendered in various sizes using the size prop.

```tsx
<div>
  <PlaylistAddIcon size={16} />
  <PlaylistAddIcon size={24} />
  <PlaylistAddIcon size={32} />
  <PlaylistAddIcon size={48} />
</div>
```

### Icon with Custom Colors

Customize the icon color using the color prop.

```tsx
<div>
  <PlaylistAddIcon color="var(--colors-primary-default)" />
  <PlaylistAddIcon color="var(--colors-secondary-default)" />
  <PlaylistAddIcon color="var(--colors-status-positive-default)" />
  <PlaylistAddIcon color="var(--colors-text-secondary)" />
</div>
```
