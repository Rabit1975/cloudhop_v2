An individual music track search result component that visually represents a `SearchResult` entity, including its title, description (artist and album), thumbnail, and type. The entire card is clickable to navigate to the track's playback page.

### Basic Music Track Result

An example of a basic music track result showing title, artist, and duration

```tsx
<MusicTrackSearchResult
  result={{
    id: 'result-1',
    content: {
      id: 'track-1',
      title: 'Neon Nebula',
      artist: 'Cosmic Beats',
      thumbnailUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_abstract_digital_music_visuali_0_1770834026837.png',
      sourceType: 'upload',
      duration: 245,
    },
    relevanceScore: 0.95,
  }}
/>
```
