A component for a MusicStudioProject entity, which encapsulates details about a music production project, including its name, owner, last edited timestamp, and a collection of music tracks (layers).

## Get started

This component provides a robust way to manage music studio projects, supporting serialization and de-serialization for seamless data transfer between backend and frontend.

### Creating a MusicStudioProject

This example demonstrates how to create a MusicStudioProject instance from a plain object.

```ts
import { MusicStudioProject } from './music-studio-project.js';
const projectPlainData = {
  id: 'proj-456',
  name: 'Morning Jam Session',
  ownerId: 'user-abc',
  lastEditedAt: new Date().toISOString(),
  layers: [
    {
      id: 'track-01',
      name: 'Drums Loop',
      durationSeconds: 60,
      path: '/audio/drums.wav',
      startTime: 0,
      volume: 1.0,
      pan: 0.0,
      isEnabled: true,
      // Assume other MusicTrackPlain properties are here
    },
    {
      id: 'track-02',
      name: 'Guitar Riff',
      durationSeconds: 30,
      path: '/audio/guitar.wav',
      startTime: 10,
      volume: 0.8,
      pan: 0.2,
      isEnabled: true,
    },
  ],
};
const project = MusicStudioProject.from(projectPlainData);
console.log(project.name);
```

### Serializing a MusicStudioProject

This example demonstrates how to convert a MusicStudioProject instance back into a plain object for storage or transmission.

```ts
import { MusicStudioProject } from './music-studio-project.js';
const projectPlainData = {
  id: 'proj-789',
  name: 'Evening Chillout',
  ownerId: 'user-xyz',
  lastEditedAt: new Date().toISOString(),
  layers: [
    {
      id: 'track-03',
      name: 'Synth Pad',
      durationSeconds: 120,
      path: '/audio/synth.wav',
      startTime: 0,
      volume: 0.7,
      pan: 0.0,
      isEnabled: true,
    },
  ],
};
const project = MusicStudioProject.from(projectPlainData);
const plainObject = project.toObject();
console.log(plainObject.id);
```
