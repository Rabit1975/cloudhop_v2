A component for a MusicTrack entity, supports serialization and de-serialization of MusicTrack objects and can be used from both backend and frontend.

## Get started

Install and use the component in your code:

```ts
import { MusicTrack } from '@cloudhop-v2/music-streaming.entities.music-track';
const track = MusicTrack.from({
  id: 'track-01',
  title: 'Neon Nights',
  artist: 'Cyberwave',
  duration: 215,
  sourceType: 'youtube',
  sourceIdentifier: 'dQw4w9WgXcQ',
  isPublic: true,
  createdAt: '2023-10-27T10:00:00Z',
});
```
### Creating a MusicTrack instance
This example demonstrates how to create a MusicTrack object from a plain JavaScript object.
```ts
import { MusicTrack } from '@cloudhop-v2/music-streaming.entities.music-track';
const trackData = {
  id: 'new-track-id',
  title: 'Morning Groove',
  artist: 'Sunrise Beats',
  duration: 200,
  sourceType: 'upload',
  sourceIdentifier: 'https://example.com/morning-groove.mp3',
  isPublic: true,
  createdAt: '2024-01-01T08:00:00Z',
};
const musicTrack = MusicTrack.from(trackData);
console.log(musicTrack.title);
```
### Serializing a MusicTrack instance
This example shows how to convert a MusicTrack object back into a plain JavaScript object for data transfer or storage.
```ts
import { MusicTrack } from '@cloudhop-v2/music-streaming.entities.music-track';
const trackData = {
  id: 'another-track-id',
  title: 'Evening Chill',
  artist: 'Relaxation Zone',
  duration: 320,
  sourceType: 'youtube',
  sourceIdentifier: 'abcDEF12345',
  isPublic: false,
  createdAt: '2024-01-02T19:00:00Z',
};
const musicTrack = MusicTrack.from(trackData);
const plainObject = musicTrack.toObject();
console.log(plainObject.duration);
```
