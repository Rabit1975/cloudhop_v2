---
description: Displays an individual space search result.
labels: ['space', 'search', 'ui']
---

A SpaceSearchResult component displays information about a space found through a search operation. It includes the space name, description, visibility status, and an icon, all within a clickable card that navigates to the space URL.

### Public Space Result
A SpaceSearchResult component displaying a public space.

```jsx
const publicSpaceResult = {
  id: 'space-1',
  content: {
    id: 'space-1',
    name: 'CloudHop Community',
    description: 'The official community space for CloudHop users to share ideas and feedback.',
    visibility: 'public',
  },
  score: 1,
};
<div style={{ maxWidth: '340px', padding: '24px' }}>
  <SpaceSearchResult result={publicSpaceResult} />
</div>
```
