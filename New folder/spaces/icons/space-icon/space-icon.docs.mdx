## Basic Space Icon

Render the default space icon.

```tsx
<SpaceIcon />
```
## Icon with Custom Sizes
Demonstrate how to render the space icon with various custom sizes.
```tsx
<div>
  <SpaceIcon size={16} />
  <SpaceIcon size={24} />
  <SpaceIcon size={32} />
  <SpaceIcon size={48} />
</div>
```
## Icon with Custom Colors
Show the space icon rendered with different custom colors.
```tsx
<div style={{ display: 'flex', gap: '24px' }}>
  <SpaceIcon color="var(--colors-primary-default)" />
  <SpaceIcon color="var(--colors-secondary-default)" />
  <SpaceIcon color="var(--colors-status-info-default)" />
  <SpaceIcon color="var(--colors-text-secondary)" />
</div>
```

    Creative Spaces
  </span>
