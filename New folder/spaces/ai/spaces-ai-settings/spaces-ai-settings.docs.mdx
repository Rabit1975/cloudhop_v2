A dedicated AI settings component for configuring AI-related preferences within CloudHop spaces. Users can select preferred AI models, manage API keys, and adjust generation parameters for content creation and other AI-powered features.

### Default AI Settings Configuration

This example shows the basic rendering of the AI settings component, allowing users to configure their preferred AI models, API keys, and generation parameters.

```tsx
<SpacesAiSettings />
```
### Customizing AI Provider
This demonstrates how users can select different AI providers, such as OpenAI or Anthropic, to power their creative tools within spaces. The component handles the selection and state management internally.
```tsx
<SpacesAiSettings />
```
