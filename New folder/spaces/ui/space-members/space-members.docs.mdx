---
description: A component to display members of a space, including avatars, names, and detailed tooltips on hover.
labels: ['space', 'members', 'community', 'profile']
---

This component displays a list of space members, showing their avatars and names. When a user hovers over a member's avatar, a tooltip appears with more detailed information about that member. It uses `useUserProfile` to fetch member details and supports displaying a limited number of members with a counter for the rest.

### Basic space members display

Displays a list of space members with their avatars and names. Hover over an avatar to see more details.

```tsx
import { SpaceMembers } from '@cloudrabbit/spaces.ui.space-members';
// @ts-ignore The mock data here might not cover all properties required by the external UserProfile type.
const mockUsers = {
  'user-1': {
    id: 'user-1',
    username: 'alice_wonder',
    displayName: 'Alice Wonderland',
    bio: 'Digital artist and space explorer. Loving the nebula vibes!',
    imageUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_a_vibrant__diverse_group_of_yo_0_1770834035481.png',
  },
  'user-2': {
    id: 'user-2',
    username: 'bob_builder',
    displayName: 'Bob The Builder',
    bio: 'Constructing digital realities one block at a time.',
    imageUrl: 'https://storage.googleapis.com/bit-generated-images/images/image_a_vibrant__diverse_group_of_yo_0_1770834035481.png',
  },
  'user-3': {
    id: 'user-3',
    username: 'charlie_chap',

const memberIds = ['user-1', 'user-2', 'user-3'];
  <SpaceMembers members={memberIds} mockProfiles={mockUsers} />
```
