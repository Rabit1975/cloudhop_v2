---
description: A component to display a preview of an asset, showing images for image types and generic icons for others.
labels: ['asset', 'preview', 'ui']
---

This component provides a visual preview of various asset types. For images, it displays the actual image. For other asset types like documents, audio, or video, it shows a generic icon along with the asset's name and type. It supports an optional click handler for interaction.

### Image Asset Preview

Display a preview for an image asset.

```tsx
const placeholderImage = 'https://storage.googleapis.com/bit-generated-images/images/image_a_modern_and_clean_ui_design_f_0_1770833871858.png';
const imageAsset = {
  id: 'img-123',
  name: 'Design_System_v2.png',
  type: 'image/png',
  url: placeholderImage,
};
<AssetPreview asset={imageAsset} />
```

### Document Asset Preview

Display a preview for a document asset, showing a generic file icon.

```tsx
const docAsset = {
  id: 'doc-456',
  name: 'Q4_Financial_Report.pdf',
  type: 'application/pdf',
  url: '',
};
<AssetPreview asset={docAsset} />
```

### Video Asset Preview

Display a preview for a video asset, showing a generic video icon.

```tsx
const videoAsset = {
  id: 'vid-789',
  name: 'Product_Demo_Walkthrough.mp4',
  type: 'video/mp4',
  url: '',
};
<AssetPreview asset={videoAsset} />
```

### Clickable Asset Preview

An asset preview that triggers an action when clicked.

```tsx
const placeholderImage = 'https://storage.googleapis.com/bit-generated-images/images/image_a_modern_and_clean_ui_design_f_0_1770833871858.png';
const clickableAsset = {
  id: 'click-001',
  name: 'Clickable_Asset.jpg',
  type: 'image/jpeg',
  url: placeholderImage,
};
<AssetPreview 
  asset={clickableAsset} 
  onClick={(asset) => alert(`Clicked on ${asset.name}`)}
/>
