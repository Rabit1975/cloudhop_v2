---

The `SpaceCard` component is designed to present information about a single space in a clean, interactive format. It displays the space name, a brief description, its visibility status, and the number of members. The entire card is clickable, navigating users to the space's detail page.

### Basic Space Card

```tsx
import { SpaceCard } from './space-card.js';
import { Space } from '@cloudrabbit/spaces.entities.space';
const publicSpace = Space.from({
  id: 'space-nebula',
  name: 'Nebula Explorers',
  description: 'A community for those who love to explore the vast unknown of the digital universe. Join us for weekly events and discussions.',
  ownerId: 'user-1',
  members: Array(1250).fill('user'),
  visibility: 'public',
  createdAt: new Date().toISOString(),
});
<div style={{ padding: '32px', maxWidth: '360px', backgroundColor: 'var(--colors-surface-background)' }}>
  <SpaceCard space={publicSpace} />
</div>
```

### Space Card with a Custom Image

This example shows how to use a custom image for the space card.
```tsx
import { SpaceCard } from './space-card.js';
import { Space } from '@cloudrabbit/spaces.entities.space';
const customImageSpace = Space.from({
  id: 'space-forest',
  name: 'Enchanted Forest',
  description: 'Step into a magical realm where nature thrives and mysteries unfold. Discover ancient lore and hidden paths.',
  ownerId: 'elf-queen',
  members: Array(300).fill('fairy'),
  visibility: 'public',
  createdAt: new Date().toISOString(),
});
<div style={{ padding: '32px', maxWidth: '360px', backgroundColor: 'var(--colors-surface-background)' }}>
  <SpaceCard 
    space={customImageSpace} 
    imageUrl="https://storage.googleapis.com/bit-generated-images/images/image_fantasy_forest_with_glowing_0_1770834012729.png" 
  />
</div>
```

### Space Cards with Different Visibilities

This example illustrates SpaceCards representing public, private, and unlisted spaces.
```tsx
import { SpaceCard } from './space-card.js';
import { Space } from '@cloudrabbit/spaces.entities.space';
const publicSpace = Space.from({
  id: 'space-public',
  name: 'Public Plaza',
  description: 'Open to all, a place for community discussions and public announcements.',
  ownerId: 'admin',
  members: Array(1200).fill('user'),
  visibility: 'public',
  createdAt: new Date().toISOString(),
});
const privateSpace = Space.from({
  id: 'space-private',
  name: 'Secret Chamber',
  description: 'Access restricted to members only. Discussions on top secret projects.',
  ownerId: 'council',
  members: Array(50).fill('agent'),
  visibility: 'private',
  createdAt: new Date().toISOString(),
});
const unlistedSpace = Space.from({
  id: 'space-unlisted',
  name: 'Hidden Retreat',
  description: 'Not publicly discoverable, but accessible via direct link. For invited guests.',
  ownerId: 'host',
  members: Array(15).fill('guest'),
  visibility: 'unlisted',
  createdAt: new Date().toISOString(),
});
<div style={{ 
  padding: '32px', 
  display: 'grid', 
  gridTemplateColumns: 'repeat(auto-fill, minmax(300px, 1fr))', 
  gap: '24px',
  backgroundColor: 'var(--colors-surface-background)' 
}}>
  <SpaceCard space={publicSpace} />
  <SpaceCard space={privateSpace} />
  <SpaceCard space={unlistedSpace} />
</div>
