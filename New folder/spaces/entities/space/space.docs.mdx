A component for a Space entity, supports serialization and de-serialization of Space objects and can be used from
both backend and frontend.

### Creating a Space entity

This example shows how to create a new Space entity using the static from method.

```ts
import { Space } from './space.js';
const newSpace = Space.from({
  id: 'my-unique-space-id',
  name: 'My Awesome Space',
  description: 'A place for collaboration and fun',
  ownerId: 'user-alice',
  members: ['user-alice', 'user-bob'],
  visibility: 'public',
  createdAt: new Date().toISOString(),
});
console.log(newSpace);
```

### Serializing a Space entity

This example demonstrates converting a Space entity back into a plain JavaScript object.

```ts
import { Space } from './space.js';
const existingSpace = Space.from({
  id: 'existing-space-id',
  name: 'Project Alpha',
  description: 'Our secret project workspace',
  ownerId: 'user-charlie',
  members: ['user-charlie'],
  visibility: 'private',
  createdAt: '2023-01-01T10:00:00Z',
});
const plainObject = existingSpace.toObject();
console.log(plainObject);
```
