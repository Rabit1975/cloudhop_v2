The Content entity represents various types of digital content within a CloudHop space. It includes properties such as ID, associated space, title, body text, content type, creator, linked assets, and timestamps for creation and updates. This entity supports serialization and deserialization, making it suitable for both frontend and backend usage.

### Creating a Content entity

This example shows how to create a new instance of the Content entity with required properties. This uses the class constructor directly.

```ts
import { Content } from './content.js';
const newContent = new Content(
  'content-123',
  'space-abc',
  'My First Document',
  'document',
  'user-xyz',
  new Date().toISOString(),
  'This is the body of my first document.',
  []
);
console.log(newContent.title);
// => "My First Document"
```

### Converting Content to and from a plain object
This demonstrates how to convert a Content entity to a plain JavaScript object for serialization, and then back into a Content instance from that plain object. This is useful for data storage and retrieval.
```ts
import { Content } from './content.js';
const originalContent = new Content(
  'content-456',
  'space-def',
  'Image Gallery',
  'image',
  'user-lmn',
  new Date().toISOString(),
  'A collection of nature images.',
  ['asset-001', 'asset-002']
);
// Convert to a plain object
const plainObject = originalContent.toObject();
console.log(plainObject.type);
// => "image"
// Convert back to a Content instance
const recoveredContent = Content.from(plainObject);
console.log(recoveredContent.creatorId);
// => "user-lmn"
```

### Generating mock content for development and testing
This example shows how to use the provided mock utilities to generate realistic Content entities for testing or UI development without needing actual data. The mockContent function creates a single entity and mockContentList creates an array of entities.
```ts
import { mockContent, mockContentList } from './content.mock.js';
// Generate a single mock content item
const singleContent = mockContent({ title: 'Mocked Document Title' });
console.log(singleContent.title);
// => "Mocked Document Title"
// Generate a list of three mock content items
const contentItems = mockContentList(3);
console.log(contentItems.length);
// => 3
console.log(contentItems[0].type);
// => "document"
