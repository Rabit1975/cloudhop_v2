A dashboard panel component to display recent activity within spaces or a quick overview of joined spaces. It uses the PanelCard component as its container and displays a title, a brief description, and a list of recent spaces or activities, linking to their respective pages. It also includes a relevant space icon.


### Basic Activity Panel

This example demonstrates the basic usage of the Spaces Activity Panel component, displaying a list of recent spaces.

```tsx
// Example mock data (Space type would be imported in a real application)
const mockSpaces = [
  {
    id: 'space-1',
    name: 'Nebula Explorers',
    description: 'Exploring the digital universe.',
    ownerId: 'user-1',
    members: Array(120).fill('user'),
    visibility: 'public',
    createdAt: new Date().toISOString(),
  },
  {
    id: 'space-2',
    name: 'Design System Team',
    description: 'Internal design syncs.',
    ownerId: 'user-2',
    members: Array(15).fill('user'),
    visibility: 'private',
    createdAt: new Date().toISOString(),
  },
  {
    id: 'space-3',
    name: 'Game Dev Hub',
    description: 'Unity and HTML5 game showcase.',
    ownerId: 'user-3',
    members: Array(450).fill('user'),
    visibility: 'public',
    createdAt: new Date().toISOString(),
  },
];
<div style={{ padding: '40px', maxWidth: '360px', height: '500px', border: '1px solid var(--colors-border-default)', borderRadius: 'var(--borders-radius-medium)' }}>
  <SpacesActivityPanel mockData={mockSpaces} />
</div>
```
### Empty Activity Panel
This example shows the panel when no spaces or activities are available, displaying a message to the user.
```tsx
<div style={{ padding: '40px', maxWidth: '360px', height: '400px', border: '1px solid var(--colors-border-default)', borderRadius: 'var(--borders-radius-medium)' }}>
  <SpacesActivityPanel mockData={[]} />
</div>
```
